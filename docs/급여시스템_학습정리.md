# 급여 시스템 학습 정리 (참고 자료 기반)

참고하신 급여 화면(메디플로우 스타일) 자료를 바탕으로 정리한 스펙·구조 요약입니다.  
현재 newmso 급여 모듈에 적용할 때 참고용으로 사용하세요.

---

## 1. 화면/기능 구조

| 구분 | 내용 |
|------|------|
| **기간** | 월 단위 (예: 2026년 02월) |
| **탭/뷰** | **비과세** · **과세** · **근무일** · (메인: 근무일/지급/공제/차인지급액) |
| **인사정보** | 월별 인사정보 모달: 계약구분, 임금형태, 공제가족, 보육수당, 출산전후, 업무차량, 4대보험 |
| **급여항목** | 급여 항목(과세/비과세) 설정·편집 진입점 |

---

## 2. 비과세 (Non-taxable)

- **항목 예시**: 식비, 당직수당(야간), 차량유지비, 보육수당
- **직원별 입력** 후 **비과세합계** = 식비 + 당직수당 + 차량유지비 + 보육수당
- **합계 행**: 항목별 합계 + 비과세합계 총액
- **선지급 행**: 항목별 선지급액 + 비과세 선지급 합계 (정산과 별도 관리)
- 컬럼 헤더 **+ / −** 로 항목 추가·제거 또는 조정 가능한 UI

---

## 3. 과세 (Taxable)

- **항목 예시**: 기본급, 고정 연장근무수당, 고정 야간근무수당, 야간근무수당, 연장근무수당, 연장+야간근무수당, 직책수당, 연차휴가수당 → **과세합계**
- **합계 행**: 항목별 합계 + 과세합계
- **선지급 행**: 기본급·고정연장·고정야간 등 선지급액, 과세합계 선지급 (선지급은 수정 불가 안내)
- **저장하기** 로 월별 과세 데이터 저장

---

## 4. 근무일 (Work days / 시급 기반)

- **근무기간**: 해당월 (예: 02.01–02.28)
- **이번달 총 일수**: 28일
- **근무일수**: 실제 근무일/총일수 (예: 28/28, 17/28) — 미달 시 **빨간색** 강조
- **수습기간**: 없음 또는 기간 (예: 2025.11.18~2026.02.17 (100%))
- **기본 근무시간**: 209h, 196.5h 등 (월 소정근로시간)
- **통상시급**: 기본급·근무시간 기반 계산
- **고정/가변 수당**:  
  고정연장근무(금액+h), 연장근무, 고정야간근무, 야간근무, 고정휴일근무, 휴일근무, 연장+야간 등  
  → 각각 **금액 (N h)** 형태로 표기

## 5. 메인 급여 테이블 (한 화면 요약)

- **컬럼**: 선택 | 직원(아바타·이름·직급·부서) | 근무일 | 지급(비과세) | 지급(과세) | 공제 | 차인지급액 | 명세서
- **명세서**: 정산중 / 선지급
- **조정**: 지급(과세)·공제·차인지급액에 **+보정액** 표시 (예: +36,685)
- **선지급 처리**:  
  본급·공제·차인은 0원으로 두고, 괄호 안에 (선지급: 금액) 표기 + 반환 등 마이너스 금액 별도 표시
- **합계 행**: 지급(비과세)·지급(과세)·공제·차인지급액 총합 + 보정 소계
- **선지급 행**: 비과세 선지급 총액 등

## 6. 인사정보 (급여 계산에 필요한 입력)

- **계약구분**: 정규직, 기간제
- **임금형태**: 월급(임금)계약 등
- **공제가족**: 1명 등 (소득세 공제 인원)
- **보육수당, 출산전후, 업무차량**: 수당·혜택 여부/금액
- **4대보험**: 국민·건강 / 고용 / 4대보험 등 개인별 가입 조합

## 8. 공제 (Deductions)

- **탭**: 2026년 02월 **공제** (비과세·과세·근무일과 동일 월 컨텍스트)
- **컬럼**: 직원명 | **지급(과세)합계** | 국민연금 | 건강보험 | 장기요양 | 고용보험 | 소득세 | 지방소득세 | **공제합계**
- **직원별**: 지급(과세)합계 기준으로 각 공제 항목 금액 표시 (고용보험 0인 직원 등 개인별 상이 가능)
- **합계 행**: 국민연금·건강보험·장기요양·고용보험·소득세·지방소득세 항목별 합계 + 공제합계
- **선지급 행**: 공제 항목별 선지급액 + 공제합계 (선지급은 수정 불가 안내)
- **저장하기** 로 월별 공제 데이터 저장

---

## 9. 근로·임금 정보 등록 (근로&임금 정보 등록)

- **화면**: 전자계약 | 고용형태 | 입사일 | **소정근로시간** | **임금 계약기간** | **임금 세부 내용**
- **소정근로시간**: 일 N시간 N분, **월 N시간** (연필 아이콘으로 편집).  
  - **기본값 규칙**: 입력 없음(0시간)이면 **일 8시간, 주 48시간(주휴포함), 월 209시간(주휴포함)** 적용
- **임금 계약기간**: 시작일–종료일 (종료일 없으면 “-”로 무기한)
- **임금 세부 내용**: 월급 금액 + 비고 (예: “수습 3개월”)
- **고용형태**: 정규직 / 기간제 등 드롭다운
- **푸터 안내**: 계약기간·세부내용 빈 값이면 초기화; 계약종료 정보는 엑셀 다운로드에서 확인
- **액션**: 엑셀 다운로드, 이름 검색, 전체 필터, 변경 기록, 저장하기

## 10. 명세서 (통합 뷰)

- **제목**: 2026년 2월 **명세서**
- **컬럼**: 직원 | 보육수당, 고정 연장근무수당, 고정 야간근무수당, 야간근무수당, 연장근무수당, 연장+야간근무수당, 직책수당, 연차휴가수당 | 국민연금, 건강보험 | **지급합계** | **공제합계** | **차인지급액**
- **합계 행**: 수당·공제 항목별 합계 + 지급합계 + 공제합계 + 차인지급액
- **선지급 행**: 수당·공제 항목별 선지급 + 지급합계·공제합계·차인지급액 (선지급은 수정 불가, 시급 정보 없으면 근무일 팝업에서 작성 후 진행 안내)
- **UI**: Shift+마우스휠 좌우 스크롤, 엑셀 다운로드, 다운로드 기록, 급여항목(표시 항목 설정), 저장하기

## 11. 임금 계산기 (계산 규칙)

- **4대보험 계산 기준 (중요)**  
  “4대보험은 가입금액을 알 수 없으므로 **급여에서 비과세를 뺀 나머지 금액(과세표준금액)** 을 기준으로 계산합니다.”  
  → **과세표준 = 지급총액(과세+비과세) − 비과세 = 과세 소득** 기준으로 국민연금·건강보험·장기요양·고용보험 산정
- **급여(과세)**: 기본급(근무시간·금액), 연장근무수당(시간·배수 1.5 등·금액), 야간근무수당, 휴일근무수당, 기타수당 → **과세** 합계
- **급여(비과세)**: 비과세 합계 입력 → **비과세** 표시
- **공제**: 국민연금·건강보험·장기요양·고용보험·소득세·지방소득세·기타공제 (각각 수기 입력 가능) → **공제총액**
- **하단 요약**: 통상시급, 연봉, 지급총액(과세+비과세), 차인지급액(지급총액−공제총액)
- **세액 설정**: 소득세 기준 = 근로소득 간이세액표, 소득세율(100% 등), 공제대상 가족 수(본인·자녀 포함), 20세 이하 자녀 수

---

## 12. 계약 관리 (급여 연동)

- **탭**: 급여 | **계약** | 급여설정
- **좌측**: 조직도(부서별 인원), 퇴직한 구성원
- **계약 종류**: 근로계약서 | 연봉(임금)계약서 | 그 외 계약서
- **직원별**: 입사일/재직기간, “주민등록번호·주소·이메일 인증 후 계약 진행” 안내, **임금정보 등록** 버튼
- **액션**: 근로&임금 정보 등록, 계약서 작성, 계약 가이드
- **연계**: 임금정보 등록 → 근로·임금 정보(소정근로시간, 임금 계약기간, 월급/수습) 입력 → 급여·명세서 계산에 사용

---

## 13. 계약서 작성 (메디플로우 표준 양식)

### 13.1 계약서 종류 (좌측 선택 목록)

- **근로계약서** — 근로조건·소정근로시간·근무일/휴일·임금(별도 연봉계약)·연차·비밀준수 등
- **연봉계약서** — 연봉(임금) 계약기간·임금·임금지급방법 (급여와 직접 연동)
- 개인정보 수집이용 동의서, 초상권·개인정보 사용 동의서
- **피복비 환급 동의서** — 급여·비용 처리와 연계 가능
- 보안서약서, 보안서약서(퇴직)
- 근로계약해지통지서
- 인사발령 동의서
- **연장/야간/휴일 근로 동의서** — 연장·야간·휴일 수당 계산 근거

### 13.2 근로계약서 구조 (조항 예시)

| 조항 | 제목 | 급여·임금 관련 요약 |
|------|------|----------------------|
| 제1조 | 근로계약 | 계약일·기간 정함 시 만료로 자동종료 |
| 제2조 | 근무장소와 업무내용 | 장소·직무·변경 시 협의 |
| 제3조 | **소정근로시간** | 입력 필드, 협의 변경, 연장 가산 수당(사업주 승인 없는 자발 연장/야간/휴일 미인정) |
| 제4조 | 근무일/휴일 | 매주 근무일·주휴일 |
| 제5조 | **임금** | “별도 임금(연봉)계약서로 협의” → **임금별도** 선택 시 연봉계약서와 쌍으로 사용 |
| 제6조 | 연차유급휴가 | 근로기준법에 따름 |
| 제7조 | 성실 이행의무 | 근로계약·취업규칙 준수 |
| 제8조 | **비밀준수의무** | 임금내역 누설 금지·징계, 퇴직 후 업무 비밀 유지, 위반 시 민형사상 책임 |
| 제9조 | 기타 | 근로기준법·원 규정, 퇴직 30일 전 통보, **무통보 결근 시 당일 급여·해당 주 주휴수당 공제**, 퇴직금 평균임금 등 |

### 13.3 연봉(임금)계약서 구조

- **사업주/근로자**: 상호·대표자·주소 / 성명·주민등록번호(생년월일)·주소
- **[연봉계약기간]**: 시작일~종료일 (예: 2026.02.22~2027.02.21), 만료 시 성과·평가에 따른 조정, 1년 미만 시 제3조 월급 기준
- **[임금]**: 우측 **“[임금] 관리”** 버튼으로 임금 정보 입력 → 급여(월급·수당) 데이터와 연동
- **[임금지급방법]**:  
  1) 매월 1일~말일 산정, 익월 7일 통장 입금  
  2) 소득세·지방소득세·고용보험·건강보험·국민연금 공제  
  3) 중간 입·퇴사 시 월급 일할 계산  
  4) 지각·조퇴·결근 시 해당 일급·시급 공제  

### 13.4 우측 입력 패널 (공통)

- **계약 대상자\***, **사업주 대표자\***, **계약 시작일\***, **계약 종료일**, 계약 갱신일, **직무**, **작성일\***
- 생년월일·주민등록번호 포함 여부 (체크)
- **고용형태\***: 정규직 등
- **수습기간**: 수습기간 없음 / 수습기간 있음
- **임금**: **임금포함** / **임금별도** (임금별도 선택 시 근로계약서에는 “별도 연봉계약서로 협의”, 연봉계약서에서 [임금] 관리로 금액 입력)
- 연봉계약서 선택 시: **“[임금] 관리”** 버튼 → 임금(월급·수당) 상세 입력

### 13.5 급여 연동 포인트

- **임금별도** → 근로계약서 + 연봉계약서 쌍 사용, 연봉계약서의 [임금] 관리가 **실제 급여 항목(월급·수당)** 의 근거
- **[임금] 관리** → 근로·임금 정보(소정근로시간, 임금 계약기간, 월급·수습)와 동일 데이터 소스로 활용 시 급여·명세서·정산과 일관 유지
- 제9조 무통보 결근 시 **당일 급여·주휴수당 공제** → 근태차감 규칙(attendance_deduction)과 동일 로직
- 연장/야간/휴일 근로 동의서 → 연장·야간·휴일 수당 지급 근거로 명세서·정산과 연결
