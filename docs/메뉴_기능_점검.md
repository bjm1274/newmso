# MSO 메뉴별 기능 점검

## 메뉴 구조 및 현황

| 메뉴 | 컴포넌트 | 주요 기능 | 상태 |
|------|----------|-----------|------|
| 내정보 | MyPage | 프로필, 급여명세, 출퇴근, 나의할일, 증명서 | ✅ |
| 조직도 | OrgChart | 직원 트리, 회사별 필터 | ✅ |
| 채팅 | ChatView (메신저) | 1:1/그룹 채팅, 공지, 투표, 반응, 고정 | ✅ |
| AI채팅 | AIChatView | OpenAI 연동 대화 | ✅ |
| 게시판 | BoardView | 자유/공지/수술일정, 댓글, 좋아요 | ✅ |
| 알림 | NotificationInbox | 알림 목록, 읽음 처리 | ✅ |
| 전자결재 | ApprovalView (전자결재) | 연차/비품/인사/출결정정 | ✅ |
| 인사관리 | HRView | 구성원, 근무형태, 근태, 급여, 휴가, 증명서 | ✅ |
| 재고관리 | 재고관리_통합완성 | 현황, UDI, 명세서, 발주, 스캔, 촬영, 등록 | ✅ 고도화 |
| 관리자 | AdminView | 대시보드, 직원, 회사, 권한, 데이터백업, 감사로그 | ✅ |

## 주의사항
- **근태관리**: 사이드바에 없음 (인사관리 > 근태 탭에서 접근)
- **전자결재**: 전자결재.tsx 사용 (전자결재_개선.tsx는 별도 개선 버전)
