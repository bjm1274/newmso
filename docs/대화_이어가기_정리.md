## 2. 최근 구축된 기능 (2025-02 기준)

### 2.1 조직도·팀 관리
- **구조**: 병원장 → 진료부/간호부/총무부 → 각 팀
- **테이블**: `org_teams` (company_name, division, team_name)
- **관리자**: 팀 추가/삭제 (관리자 > 팀 탭)
- **파일**: `조직도그림.tsx`, `팀관리.tsx`, `구성원현황.tsx`

### 2.2 직원·권한 통합
- **통합 화면**: 관리자 > 직원·권한 (한 탭에서 직원 목록 + 역할·세부권한 + 비밀번호 설정)
- **권한 키**: mso, hr, inventory, approval, admin, mso_plus_all
- **파일**: `직원권한통합.tsx` (직원관리도구 + 권한설정도구 통합)

### 2.3 법인카드 회사별 관리
- **카드 등록**: 회사별 법인카드 등록 (카드별칭, 끝4자리, 발급사, 사용자)
- **사용내역**: 회사·카드별 필터, Supabase Realtime, CSV 일괄 가져오기
- **테이블**: `corporate_cards`, `corporate_card_transactions` (card_id FK)

## 9. 2026-02-11 현재 작업 상태 (인사·재고·스캔)

### 9.1 오늘까지 반영된 핵심 변경점

- **마이페이지(내 정보)**
  - `마이페이지` 내에서 **이메일/연락처/주소/계좌 정보**를 사용자가 직접 수정 가능.
  - 출퇴근 기록 탭에서 **정정 요청 버튼** 클릭 시, 메인 메뉴의 `전자결재` 탭이 열리도록 연동 완료.
- **전자결재 → 서식양식 + 급여명세서 서식 디자이너**
  - 관리자 `전자결재양식관리.tsx` → **`서식양식 관리`** 로 개편.
  - `system_settings` 테이블의 `form_template_designs` JSON을 기반으로 **양식/급여명세서 WYSIWYG 디자이너** 구현.
  - 제목/부제/서명 영역을 **미리보기 화면에서 마우스로 클릭한 위치에 배치**하는 기능 포함 (퍼센트 단위 좌표).
  - `급여명세서PDF.tsx` 에서 이 디자인 정보를 불러와 **동일한 레이아웃·색상으로 PDF 렌더링**.
- **인사관리 – 퇴사자 필터**
  - 인사 관리 화면에서 **퇴사자만 따로 볼 수 있는 필터** 추가(상세 구현은 인사 관련 서브 컴포넌트 참조).
- **재고 – 물품 등록**
  - `물품등록.tsx` 에 **보험코드(insurance_code), 규격(spec)** 입력 필드 추가.
  - 신규 등록 시 `stock` 컬럼을 `quantity` 값으로 **동일하게 채워 NOT NULL 오류 해결**.
  - 배정 부서 드롭다운이 `staff_members` 테이블에서 **회사 구분 없이 모든 부서명(수연의원, SY INC. 등)** 을 모아 보여주도록 수정.
- **재고 – 현황 필터**
  - `재고관리_통합완성.tsx` 의 `현황` 탭에서
    - 상단 검색창 + **부서 선택 드롭다운** 추가
    - 선택한 회사(`selectedCo`) + 부서(`selectedDept`)에 따라 재고 목록 필터링.
- **재고 – 스캔 입고 (중요 전환)**
  - 기존 **명세서 OCR + 카메라/이미지 스캔 기반 모듈은 한계(OCR 인식 불량)** 때문에 사용 중단.
  - `스캔모듈완성.tsx` 를 **의료기기 QR/바코드 전용 모듈**로 전면 재작성:
    - 스캐너가 입력하는 문자열을 **입력창에서 직접 받고, Enter 시 처리**.
    - `inventory.barcode` 또는 `inventory.udi_code` 와
      - **완전 일치**, **포함**, **역포함** 까지 허용하여 GS1 UDI 문자열도 매칭.
    - 매칭된 품목을 **스캔 목록에 누적**하고, 하단 버튼으로 `inventory`/`inventory_logs` 에 입고 반영.
  - 더 이상 **명세서 OCR·카메라 촬영·이미지 업로드**는 사용하지 않음(이 모듈에서는 완전히 제거).
- **재고 – 등록 방식 분리 (일반등록 / 엑셀등록)**
  - `재고관리_통합완성.tsx` 의 `등록` 탭:
    - 상단에 **"✏️ 일반 등록" / "📊 엑셀 일괄 등록"** 토글 버튼 추가.
    - 선택에 따라
      - `물품등록` 컴포넌트 (기존 개별 등록 폼)
      - `ExcelBulkUpload` 컴포넌트 (엑셀 파일을 통한 일괄 등록)
      를 전환해서 표시.
