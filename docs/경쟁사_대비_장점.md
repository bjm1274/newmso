# SY INC. MSO vs Flex·Clap·Shiftee 경쟁사 비교

## 경쟁사 개요

| 제품 | 특징 | 주요 고객 |
|------|------|----------|
| **Flex** (flex.team) | 범용 HR 플랫폼, 구성원·근태·계약·급여·성과·채용 | 일반 기업 |
| **Clap** (clap.company) | AI 기반 성과관리(평가·목표·원온원) 전용 | 1,000+ 팀 |
| **Shiftee** (shiftee.io) | 의료기관 교대근무·근태 통합 솔루션 | 300,000+ 사업장 |

## SY INC. MSO 우리 시스템의 장점

### 1. 통합 검색 (경쟁사 없음)
- **직원·게시글·결재·채팅** 한 번에 검색
- Flex, Shiftee, Clap 모두 메뉴별 분리 검색만 제공
- 업무 상황에서 "한 곳에서 빠르게 찾기" 가능

### 2. 중앙 문서 보관·버전 관리 (경쟁사 약함)
- **규정·양식·계약서** 한 곳에 보관, **버전 이력** 자동 관리
- Flex/Shiftee: 계약서 보관은 있으나 버전 관리·규정 통합은 불명확
- Clap: 문서 보관 기능 없음 (성과관리 전용)

### 3. MSO·멀티병원 전용 설계
- **박철홍정형외과, 수연의원, SY INC.** 등 **여러 병원을 하나의 플랫폼**에서 관리
- Flex/Shiftee: 멀티테넌트 지원은 있으나 **의료 MSO 전용** 아님
- 회사별 팀 구조, 사업체 필터, 권한 분리 등 MSO 시나리오에 최적화

### 4. 재고·UDI 관리 (의료기관 특화)
- **재고관리**, **UDI 스캔**, 명세서·발주 통합
- Flex, Clap: 재고 기능 없음
- Shiftee: 근태·스케줄 중심, 재고 없음

### 5. 법인카드 회사별 관리
- 회사별 법인카드 등록, 사용 내역 실시간/CSV 반영
- 경쟁사에서 명시적 제공 사항 없음

### 6. 내부 협업 통합
- **채팅·게시판·알림** 시스템 내 통합
- Flex: 별도 협업 툴 연동 위주
- Shiftee: 사내 메시지 있으나 협업 기능은 제한적

### 7. 외부 서비스 바로가기
- KM Park, U+ 웹팩스 등 **의료 MSO 전용 링크** 통합
- 업무 컨텍스트에 맞는 단일 접속 경로 제공

### 8. 완전한 커스터마이징
- 자체 개발 시스템으로 **요구사항별 확장** 가능
- Flex/Shiftee: SaaS 고정 기능 위주
- Clap: AI 커스터마이징 있으나 성과관리 한정

### 9. 비용 구조
- Flex: 월 1인당 약 6,000원 + 기능별 추가
- Shiftee: 사업장/인원 단위 과금
- **MSO**: 자체 인프라(Supabase) 사용 시 **인원당 고정 비용 없음** (DB·호스팅만 부담)

### 10. 데이터 소유권
- Supabase 사용 시 **데이터가 우리 DB**에 저장
- SaaS 경쟁사는 클라우드 벤더 DB에 데이터 보관
