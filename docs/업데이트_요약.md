# SY INC. MSO 시스템 - 업데이트 요약

> 최근 적용된 수정·추가 사항 정리 (2025-02 기준)

---

## 1. 전자결재

### 수리요청서 양식 추가
- **위치**: 전자결재 → 작성하기 → 양식 탭에 **수리요청서** 추가
- **입력 항목**: 장비/시설명, 위치, 희망 수리일, 긴급도, 요청 내용
- **파일**: `app/main/기능부품/전자결재서브/수리요청서양식.tsx` (신규)

### 관리자 메뉴에서 전자결재 양식 추가/수정
- **위치**: 관리자 → **전자결재 양식** 탭
- **기능**: 추가 양식 이름·코드(slug) 등록, 수정, 비활성화, 삭제
- **DB**: `approval_form_types` 테이블 사용 (`docs/미적용_SQL_통합.sql` 또는 `supabase_migrations/approval_form_types.sql` 적용 필요)
- **파일**: `app/main/기능부품/관리자전용서브/전자결재양식관리.tsx` (신규)

### 실시간 반영
- `approvals` 테이블 Realtime 구독 → 기안/승인 시 목록 자동 갱신

---

## 2. AI 채팅

- **한도 초과(429) 시**: "한도 초과" 문구 최소화, **기본 안내 문구**를 메인 응답으로 표시
- **시스템 프롬프트 강화**: 메뉴 구조(조직도, 전자결재·양식, 인사·재고·근태·내 정보·관리자 등)와 답변 원칙 추가
- **데모 응답**: 전자결재(수리요청 포함), 재고, 내 정보(할일·사진) 관련 안내 추가

---

## 3. 내 정보

### 사진 등록
- ID 복구 시 `erp_user`도 함께 갱신
- 업로드 전 ID 없으면 이름으로 한 번 더 복구 시도 후 `erp_user`에서 ID 사용
- **ID 없는 계정**(예: MSO 관리자): 사진 버튼 대신 "직원 계정 로그인 시 사진 등록 가능" 문구 표시

### 할일
- **문구 변경**: "사용자 ID를 복구하고 있습니다…" → **"할일은 직원 계정(이름으로 로그인)으로 이용해 주세요. MSO 관리자 계정은 이 기능을 사용할 수 없습니다."**
- 복구 시 `erp_user` 동기화
- **실시간 반영**: `todos` 테이블 Realtime 구독 → 할일 추가/수정/삭제 시 목록 자동 갱신

### 알림 탭 추가
- 내 정보 탭에 **🔔 알림** 추가 → 기존 알림 센터(전체/안읽음, 읽음 처리) 사용

## 4. 로그인

- **MSO 관리자(100 / MSO관리자)**: DB에 `name = 'MSO관리자'`인 직원이 있으면 해당 행의 **id**를 넣어 저장 → 할일·사진 등 이용 가능

## 5. 테마 모드 제거

- PC 사이드바 하단 **테마(☀️/🌙) 버튼** 제거
- 모바일 더보기 메뉴에서 **테마** 행 제거
- **파일**: `app/main/기능부품/조직도서브/조직도측면창.tsx` (useTheme 제거)

---

## 6. 채팅

- **전직원 공지방** 제거: 좌측 채팅 목록에서 "📢 전직원 공지방" 항목 삭제
- **채팅방 공지 블록** 제거: 본문 상단 "📢 채팅방 공지"(고정 메시지 영역) 삭제
- **메시지 상하 폭 축소**: 버블 `p-4` → `px-3 py-2`, 목록 간격 `space-y-6` → `space-y-3`
- 채팅방 없을 때 "채팅방을 선택하세요" 표시, 채팅방 목록 로드 시 첫 번째 방 자동 선택

---

## 7. 알림 메뉴 통합

- **사이드바**: **알림** 메뉴 제거
- **내 정보**: 알림 탭 추가 (위 3번 참고)
- **우측 상단 전역 알림**
  - **파일**: `app/components/GlobalNotificationBell.tsx` (신규)
  - 상단바 우측에 🔔 버튼 + 안 읽은 개수 뱃지
  - 클릭 시 최근 알림 8건 드롭다운, "전체 보기" 시 내 정보 → 알림 탭으로 이동
  - `notifications` Realtime으로 실시간 갱신

## 8. 실시간 업데이트 적용 현황

| 화면 | 테이블 | 내용 |
|------|--------|------|
| 알림 (내 정보) | `notifications` | 새 알림/읽음 시 목록 자동 갱신 |
| 우측 상단 알림 벨 | `notifications` | 개수·목록 실시간 갱신 |
| 전자결재 | `approvals` | 기안/승인 시 기안함·결재함 갱신 |
| 할일 | `todos` | 추가/수정/삭제/완료 시 목록 갱신 |
| 게시판 | `board_posts` | 새 글·수정·삭제·좋아요 시 목록 갱신 |
| 채팅 | `messages`, `chat_rooms` 등 | 기존 Realtime 유지 |
| 법인카드 사용내역 | `corporate_card_transactions` 등 | 기존 Realtime 유지 |

※ Supabase Dashboard → Database → Replication 에서 해당 테이블이 활성화되어 있어야 합니다.
