# NEWMSO 배포용 공지문 초안

## 1. 일반 사원 공지문

### 사내 공지형

안녕하세요.  
NEWMSO 사용 안내서를 배포드립니다.

이번 안내서는 현재 사용 중인 원본 프로그램 기준으로 정리되었으며, 사원분들이 자주 사용하는 기능을 중심으로 정리했습니다.

- 내정보: 출퇴근, 급여, 증명서, 서류 제출, 알림
- 채팅: 공지 확인, 1:1/그룹 대화
- 게시판: 공지사항, 자유게시판, 경조사 소식
- 전자결재: 연차, 출결정정, 물품신청, 업무기안

첨부 문서:
- `01_employee_manual_ko.md`

메뉴가 보이지 않거나 사용 권한이 없는 경우에는 부서장 또는 관리자에게 문의해 주세요.  
급여, 증명서, 인사 정보 오류는 인사 담당자 또는 관리자에게 알려주시면 됩니다.

### 메신저/카카오톡형

NEWMSO 사원용 사용 안내서를 공유드립니다.  
출퇴근, 급여, 증명서, 서류제출, 전자결재 사용 방법이 정리되어 있습니다.

첨부:
- `01_employee_manual_ko.md`

메뉴가 안 보이거나 사용이 막히는 경우에는 부서장 또는 관리자에게 문의 부탁드립니다.

## 2. 부서장 공지문

### 사내 공지형

안녕하세요.  
부서장급 사용자를 위한 NEWMSO 운영 안내서를 공유드립니다.

이번 문서는 팀장, 실장, 부장, 간호과장 등 승인과 팀 운영을 함께 담당하는 사용자 기준으로 정리했습니다.

주요 내용:
- 내정보 대시보드 활용
- 전자결재 승인 처리
- 채팅 공지와 팀 커뮤니케이션
- 인사관리 활용
- 재고 부족 및 발주 대응
- 추가기능 내 부서 운영 기능

첨부 문서:
- `02_manager_manual_ko.md`

부서별 권한 차이로 인해 실제 보이는 메뉴는 다를 수 있습니다.  
권한 또는 운영 기준 변경이 필요한 경우 관리자에게 요청해 주세요.

### 메신저/카카오톡형

부서장용 NEWMSO 안내서를 공유드립니다.  
승인, 팀 운영, 재고/근태 점검, 공지 전달 기준이 정리되어 있습니다.

첨부:
- `02_manager_manual_ko.md`

부서별로 보이는 메뉴가 다를 수 있으니, 권한 조정이 필요하면 관리자에게 요청 부탁드립니다.

## 3. 관리자 공지문

### 사내 공지형

안녕하세요.  
관리자용 NEWMSO 운영 설명서를 공유드립니다.

이번 문서는 현재 원본 프로그램 기준으로 작성되었으며, 관리자 메뉴의 실제 구조와 운영 주의사항을 중심으로 정리했습니다.

주요 내용:

- 일일/주간 운영 체크리스트

### 메신저/카카오톡형

관리자용 NEWMSO 운영 안내서를 공유드립니다.  
관리자 탭 설명, 고위험 기능 주의사항, 운영 체크리스트가 포함되어 있습니다.

## 4. 마스터 공지문

### 사내 공지형

안녕하세요.  
최고권한 운영자를 위한 NEWMSO 마스터용 설명서를 공유드립니다.

현재 원본 프로그램 기준으로 마스터 전용 별도 UI는 없으며, 마스터는 관리자 기능 전체를 최고 권한 범위로 운영하는 구조입니다.  
이번 문서는 기능 사용법보다 운영 기준과 통제 원칙에 초점을 맞춰 정리했습니다.

주요 내용:
- 최고 권한 운영 범위
- 권한 체계 통제 기준
- 감사로그 및 접근감사로그 활용
- 백업, 초기화, 양식/서식 변경 승인 원칙
- 일일/주간/월간 점검 항목

첨부 문서:
- `04_master_manual_ko.md`
- 필요 시 전체 기준 문서 `00_system_overview_ko.md`

운영 사고 대응과 권한 통제 기준 문서로 활용해 주시면 됩니다.

### 메신저/카카오톡형

마스터용 NEWMSO 운영 기준 문서를 공유드립니다.  
최고권한 운영 범위, 권한 통제, 감사 대응, 백업/초기화 기준이 정리되어 있습니다.

첨부:
- `04_master_manual_ko.md`

운영 정책과 고위험 작업 승인 기준 문서로 사용하시면 됩니다.
