# NEWMSO 부서장용 기능 설명서

## 1. 문서 목적

이 문서는 팀장, 실장, 부장, 간호과장, 병원장 등 부서장급 사용자를 위한 설명서입니다.  
부서장은 일반 사원 기능에 더해 승인, 팀 운영, 공지 전달, 재고/근태 점검 역할을 함께 수행합니다.

## 2. 부서장이 중점적으로 사용하는 메뉴

| 메뉴 | 주 용도 |
| --- | --- |
| 내정보 | 팀 현황 요약, 승인 대기, 재고 부족 확인 |
| 전자결재 | 팀원 결재 승인 |
| 채팅 | 팀 소통, 공지 전달 |
| 게시판 | 부서 공지와 일정 확인 |
| 인사관리 | 팀원 현황, 근태, 휴가, 문서 관리 |
| 재고관리 | 재고 부족, 발주, 실사 확인 |
| 추가기능 | 직원평가, 마감보고, 인계노트 등 부서 운영 보조 |

## 3. 내정보 대시보드

부서장급은 `내정보` 첫 화면에서 일반 직원보다 더 많은 요약 정보를 보게 됩니다.

대표 항목

- 팀 인원 수
- 결재 대기 건수
- 오늘 출근 현황
- 재고 부족 건수

이 화면은 하루 시작 전 점검판으로 활용하면 좋습니다.

## 4. 전자결재 운영

부서장이 가장 자주 쓰는 메뉴 중 하나가 전자결재입니다.

### 결재함

- 팀원이 올린 연차, 출결정정, 물품신청 등을 검토합니다.
- 승인, 반려, 보류 판단을 합니다.

### 자주 처리하는 문서

| 문서 | 부서장 관점 |
| --- | --- |
| 연차/휴가 | 팀 운영에 문제 없는지 확인 후 승인 |
| 출결정정 | 근태 오류 여부 확인 |
| 물품신청 | 재고와 필요성 검토 |
| 수리요청서 | 부서 설비, 장비 이슈 확인 |
| 업무기안 | 부서 운영 관련 검토 |

### 주의할 점

- 연차 승인과 출결정정은 실제 근태 데이터에 반영됩니다.
- 물품신청은 구매 흐름과 연결될 수 있습니다.
- 현재 UI상 양식빌더, 직인관리 탭이 보일 수 있어도 일반 부서장은 개인업무 중심으로 사용하는 것이 맞습니다.

## 5. 채팅과 공지 운영

부서장은 단순 대화보다 전달과 정리가 중요합니다.

주요 활용

- 팀 공지 전달
- 그룹 채팅 운영
- 파일 공유
- 중요 메시지 북마크
- 공지 등록

현재 코드상 `대표`, `부장`, `팀장`, `실장`, `병원장`, `이사`, `본부장`, `총무부장`, `진료부장`, `간호부장`은 공지메시지 방 작성 권한을 가질 수 있습니다.

## 6. 인사관리 활용

부서장은 인사관리에서 팀 단위 운영 정보를 많이 확인하게 됩니다.

### 인력관리 성격

- 구성원현황
- 인사발령
- 포상/징계
- 교육관리
- 조직도편집기
- 스킬매트릭스
- 오프보딩

### 근태 · 급여 성격

- 근태관리메인
- 시프트캘린더
- 근무표자동편성
- 휴가관리메인
- 급여관리
- 간호근무표
- 연차소멸알림
- 공휴일달력
- 지각조퇴분석
- 근무형태변경이력
- 조기퇴근감지

### 복무 · 문서 성격

- 건강검진관리
- 경조사관리
- 면허자격증관리
- 의료기기점검
- 비품장비대여관리
- 사고보고서
- 계약관리
- 문서보관함
- 증명서발급
- 스마트서류제출

현재 인사관리 내부 탭은 권한에 따라 달라질 수 있으므로, 모든 부서장에게 같은 화면이 열리지는 않을 수 있습니다.

## 7. 재고관리 활용

재고 권한이 있는 부서장은 아래 화면을 자주 확인합니다.

- 현황
- 이력
- 발주
- 거래처
- 재고실사
- 유통기한 알림
- 재고이관
- 수요예측

주요 용도

- 부족 재고 확인
- 발주 필요 품목 점검
- 유통기한 임박 품목 확인
- 실사 결과 반영

## 8. 추가기능 활용

부서장에게 유용한 추가기능은 아래와 같습니다.

| 기능 | 활용 예 |
| --- | --- |
| 인계노트 | 교대/인수인계 메모 확인 |
| 마감보고 | 일일 마감 상태 점검 |
| 직원평가 | 팀원 평가 기록 |
| 근무현황 | 현재 근무 상태 확인 |
| 부서별 재고 | 부서별 물품 상태 확인 |

## 9. 부서장이 자주 하는 업무 흐름

### 팀원 연차 승인

1. `내정보` 또는 `전자결재`에서 대기 건 확인
2. `결재함` 진입
3. 연차 요청 검토
4. 승인 또는 반려

### 부족 재고 확인

1. 대시보드에서 부족 건수 확인
2. `재고관리 > 현황` 또는 `발주` 이동
3. 필요 시 발주 또는 물품신청 진행

### 팀 공지 전달

1. 공식 공지 여부 판단
2. 공식 전달이면 공지방 또는 게시판 사용
3. 실무 협업성 전달은 그룹 채팅 사용

## 10. 운영 시 주의사항

- 부서장의 승인 행위는 실제 근태, 휴가, 구매 흐름에 연결됩니다.
- 인사 정보와 평가 정보는 민감 정보이므로 외부 공유를 금지해야 합니다.
- 현재 추가기능 일부 카드는 부서나 직책에 따라 제한됩니다.
- 인사관리 메뉴가 많기 때문에, 필요한 탭을 즐겨찾기나 반복 사용 흐름으로 익혀 두는 것이 좋습니다.

## 11. 문의 기준

| 상황 | 문의 대상 |
| --- | --- |
| 승인 권한 문제 | 관리자 |
| 팀원 인사/근태 데이터 오류 | 인사 담당자 또는 관리자 |
| 재고/발주 권한 문제 | 재고 담당자 또는 관리자 |
| 공지방 권한 문제 | 관리자 |
| 시스템 전체 설정 문제 | 관리자 또는 마스터 운영자 |
