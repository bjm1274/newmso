# NEWMSO 전체 메뉴 상세 설명서

## 1. 문서 목적

이 문서는 `C:\Users\baek_\newmso` 원본 프로그램 기준으로,  
현재 실제 화면에 연결된 메뉴를 `모메뉴 -> 서브메뉴 -> 내부 탭 -> 기능 설명` 순서로 최대한 상세하게 정리한 메뉴 사전입니다.

이 문서는 기존의 간단한 설명서보다 더 자세한 구조 설명을 위한 기준 문서이며,  
직원 교육용 PPT, 부서장용 안내서, 관리자 운영 자료를 만들 때 기준 문서로 함께 쓰는 것이 좋습니다.

## 2. 현재 내비게이션을 읽는 방법

현재 원본 프로그램은 메뉴 구조가 아래 4단계로 섞여 있습니다.

1. 메인 사이드바 메뉴
2. 사이드바 플라이아웃 서브메뉴
3. 화면 내부 상단 탭
4. 화면 내부 보조 탭

즉, 사용자는 단순히 `메인 메뉴 1개`만 누르는 것이 아니라,  
어떤 화면은 `메인 메뉴 -> 서브메뉴 -> 내부 탭`까지 들어가야 실제 기능에 도달합니다.

대표 예시

- 전자결재
  - 메인 메뉴: 전자결재
  - 서브메뉴: 기안함 / 결재함 / 작성하기 / 캘린더 / 양식빌더 / 서명관리 / 직인관리
  - 내부 탭: 작성하기 안에서 문서 유형 선택

- 인사관리
  - 메인 메뉴: 인사관리
  - 내부 업무 공간: 인력관리 / 근태 · 급여 / 복지 · 문서
  - 내부 기능 탭: 구성원현황, 급여, 연차/휴가, 계약 등
  - 내부 보조 탭: 근태 분석 탭, 급여 유틸 탭, 계약 자동생성 탭

## 6. 이 문서를 실제 배포에 활용하는 방법

- 직원 교육 때는 `내정보`, `채팅`, `게시판`, `전자결재`만 먼저 추려서 씁니다.
- 부서장 교육 때는 `전자결재`, `인사관리`, `재고관리`를 같이 설명합니다.
- 관리자 교육 때는 `관리자` 메뉴만 따로 떼서 설명하지 말고, `인사관리`, `재고관리`, `전자결재`와 연결해서 설명해야 합니다.
- 마스터 교육 때는 기능 설명보다 `통제 기준`, `권한 설계`, `변경 승인 절차`를 같이 설명해야 합니다.
