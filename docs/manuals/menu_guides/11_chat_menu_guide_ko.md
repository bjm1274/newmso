# NEWMSO 채팅 상세 설명서

## 1. 메뉴 개요

`채팅`은 공지 전달, 1:1 대화, 그룹 협업을 처리하는 실시간 소통 화면입니다.
별도 서브메뉴가 없는 대신, 하나의 큰 화면 안에 방 목록과 메시지 기능이 모두 들어 있습니다.

## 2. 진입 방법

1. 좌측 사이드바에서 `채팅`을 누릅니다.
2. 좌측 방 목록에서 대화방을 선택합니다.

## 3. 화면 구조

채팅 화면은 크게 4개 영역으로 나뉩니다.

1. `채팅방 목록`
2. `대화 영역`
3. `입력 영역`
4. `부가 기능 영역`

## 4. 주요 기능

### 방 목록

- 공지 메시지방
- 1:1 대화방
- 그룹 대화방
- 즐겨찾기 또는 고정 방

### 대화 기능

- 텍스트 메시지 전송
- 이미지, 동영상, 파일 첨부
- 멘션
- 메시지 반응
- 북마크
- 상단 공지 등록
- 메시지 전달
- 스레드 보기
- 투표 생성 및 집계

### 업무 연결 기능

- 연차/휴가 요청 초안 생성
- 발주 요청 초안 생성
- 채팅 내용에서 전자결재 흐름으로 연결

## 5. 공지 메시지방

`공지메시지`는 일반 대화방과 다르게 공식 안내용 성격이 강합니다.

현재 코드 기준 공지방 작성 가능 직책은 다음과 같습니다.

- 대표
- 부장
- 팀장
- 실장
- 병원장
- 이사
- 본부장
- 총무부장
- 진료부장
- 간호부장

## 6. 대표 사용 순서

### 1:1 메시지 보내기

1. `채팅`을 엽니다.
2. 방 목록에서 대상 직원을 찾습니다.
3. 대화방을 선택합니다.
4. 입력창에 메시지를 작성합니다.
5. 필요하면 파일을 첨부한 뒤 전송합니다.

### 그룹 대화방에서 파일 공유하기

1. 그룹 대화방을 엽니다.
2. 입력 영역의 파일 첨부 기능을 누릅니다.
3. 파일을 선택합니다.
4. 설명 메시지를 함께 입력합니다.
5. 전송 후 팀원 확인 여부를 점검합니다.

### 채팅에서 전자결재 초안 만들기

1. 관련 대화방을 엽니다.
2. 업무 요청 내용을 정리합니다.
3. 슬래시 명령 또는 연결 기능을 사용합니다.
4. `연차 요청` 또는 `발주 요청` 초안을 생성합니다.
5. 생성된 전자결재 화면에서 결재선을 검토하고 제출합니다.

## 7. 주의사항

- 공지방은 일반 대화방처럼 사적인 대화에 사용하지 않습니다.
- 메시지 첨부 파일은 팀 내 공유 자료가 될 수 있으므로 민감 정보는 주의해야 합니다.
- 채팅은 빠른 협업 도구이지만, 공식 승인과 기록이 필요한 업무는 전자결재로 마무리해야 합니다.
