# NEWMSO 전자결재 세부 설명서 3
## 양식빌더, 서명관리, 직인관리

## 1. 문서 목적

이 문서는 `전자결재` 안에 보이는 관리성 도구를 설명합니다.
현재 원본 UI에서는 일부 계정에 이 탭들이 보일 수 있지만, 실제 운영상 일반 직원보다 관리자 또는 승인권자용에 가깝습니다.

## 2. 포함 탭

1. `양식빌더`
2. `서명관리`
3. `직인관리`

## 3. 양식빌더

### 핵심 역할

- 전자결재 양식 설계
- 필드 추가/삭제
- 저장된 양식 목록 관리

### 현재 확인 가능한 빌더 성격

원본 코드 기준으로 전자결재 영역에는 커스텀 양식 설계 기능이 존재합니다.
필드 유형 예시는 아래와 같습니다.

- 텍스트
- 숫자
- 날짜
- 선택
- 체크박스
- 텍스트영역
- 파일첨부
- 서명란

### 사용 순서

1. `양식빌더`를 엽니다.
2. 양식 이름을 입력합니다.
3. 필요한 필드를 추가합니다.
4. 순서와 필수 여부를 조정합니다.
5. 미리보기 후 저장합니다.

## 4. 서명관리

### 핵심 역할

- 개인 전자서명 등록
- 서명 이미지 교체 또는 관리

### 사용 목적

- 결재 문서에 들어가는 개인 서명을 준비하는 기능입니다.

## 5. 직인관리

### 핵심 역할

- 부서 또는 회사 직인 이미지 관리
- 문서에 들어가는 공식 도장 관리

### 사용 목적

- 일반 직원보다는 관리자, 운영자, 공식 문서 담당자에게 필요한 기능입니다.

## 6. 운영상 주의사항

- `양식빌더`, `서명관리`, `직인관리`는 사용자 교육 시 일반 직원용 문서와 분리하는 것이 좋습니다.
- 잘못된 양식 수정은 여러 문서 작성 흐름에 영향을 줄 수 있습니다.
- 직인 변경은 공식 문서 효력과 연결될 수 있으므로 변경 이력 관리가 필요합니다.

## 7. 권장 배포 대상

- 관리자
- 승인권자
- 전자결재 운영 담당자
