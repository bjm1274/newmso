# NEWMSO 게시판 상세 설명서

## 1. 메뉴 개요

`게시판`은 공지와 커뮤니티, 일정형 게시판을 함께 관리하는 화면입니다.
현재 원본 UI는 사이드바 빠른 메뉴와 실제 내부 게시판 수가 다르기 때문에, 이 차이를 알고 사용하는 것이 중요합니다.

## 2. 진입 방법

1. 좌측 사이드바에서 `게시판`을 누릅니다.
2. 플라이아웃에서 빠른 게시판을 선택하거나, 게시판 화면 내부에서 다른 게시판으로 이동합니다.

## 3. 사이드바 빠른 메뉴

현재 빠른 메뉴는 아래 5개입니다.

1. `공지사항`
2. `자유게시판`
3. `경조사`
4. `수술일정`
5. `MRI일정`

## 4. 실제 내부 게시판 전체 목록

게시판 화면 안에서는 아래 7개 게시판을 사용할 수 있습니다.

1. `공지사항`
2. `자유게시판`
3. `익명소리함`
4. `경조사`
5. `수술일정`
6. `MRI일정`
7. `직원제안함`

즉, `익명소리함`과 `직원제안함`은 빠른 메뉴에는 바로 보이지 않지만 내부 게시판 목록에서는 사용할 수 있습니다.

## 5. 게시판별 기능 설명

### 공지사항

- 회사 공식 공지를 올립니다.
- 직원 알림과 연결될 수 있습니다.
- 첨부파일을 함께 올릴 수 있습니다.

### 자유게시판

- 직원 간 자유 게시글을 올립니다.
- 커뮤니티 용도로 사용합니다.

### 익명소리함

- 익명 의견 수집용 게시판입니다.
- 현재 코드 기준으로 관리자 또는 인사 권한이 아니면 목록이 제한적으로 보일 수 있습니다.

### 경조사

- 경조사 소식을 공유합니다.
- 인사관리 메뉴의 `경조사 지원 관리`와는 역할이 다릅니다.

### 수술일정

- 일정형 입력 필드가 포함된 게시판입니다.
- 날짜, 시간, 환자, 차트번호 등 운영 일정 정보를 다룹니다.

### MRI일정

- MRI 일정 입력용 게시판입니다.
- 일정형 템플릿과 필터를 사용할 수 있습니다.

### 직원제안함

- 개선 의견이나 제안을 접수합니다.

## 6. 대표 사용 순서

### 공지사항 작성

1. `게시판`을 엽니다.
2. `공지사항` 게시판으로 이동합니다.
3. 새 글 작성 버튼을 누릅니다.
4. 제목과 내용을 작성합니다.
5. 필요한 경우 첨부파일을 추가합니다.
6. 등록 후 직원 공지 여부를 확인합니다.

### 자유게시판 글 확인

1. `게시판`을 엽니다.
2. `자유게시판`을 선택합니다.
3. 목록에서 글을 고릅니다.
4. 본문과 첨부파일을 확인합니다.

### 수술일정 입력

1. `게시판`을 엽니다.
2. `수술일정` 게시판을 선택합니다.
3. 일정 입력 양식을 엽니다.
4. 날짜, 시간, 환자명, 차트번호 등 필요한 필드를 입력합니다.
5. 등록 후 부서 필터로 조회합니다.

## 7. 주의사항

- `경조사`는 게시판 메뉴에서는 소식 공유용입니다.
- 인사관리 메뉴의 `경조사`는 지원금/복지 관리 성격이라 다른 기능입니다.
- 빠른 메뉴에 없는 게시판은 화면 내부 목록에서 찾아야 합니다.
