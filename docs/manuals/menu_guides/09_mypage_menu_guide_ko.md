# NEWMSO 내정보 상세 설명서

## 1. 메뉴 개요

`내정보`는 직원이 로그인 후 가장 자주 사용하는 개인 업무 시작 화면입니다.
급여 확인, 출퇴근 기록, 증명서 신청, 서류 제출, 알림 확인이 이 메뉴에 모여 있습니다.

## 2. 진입 방법

1. 좌측 사이드바에서 `내정보`를 누릅니다.
2. 화면 상단 또는 내부 탭에서 필요한 항목을 선택합니다.

## 3. 탭 구조

현재 원본 기준 탭은 아래와 같습니다.

1. `프로필`
2. `출퇴근`
3. `할 일`
4. `증명서`
5. `급여`
6. `서류제출`
7. `알림`

## 4. 탭별 기능 설명

### 4-1. 프로필

- 내 기본 인적사항과 직무 정보를 확인합니다.
- 역할별 대시보드 카드가 표시됩니다.
- 즐겨찾기 바로가기를 사용할 수 있습니다.
- 개인 정보 수정 기능이 연결됩니다.
- 전자서명이나 계약 관련 모달이 이 영역에서 열릴 수 있습니다.

### 4-2. 출퇴근

- 오늘의 출근/퇴근 기록을 확인합니다.
- 최근 출결 상태를 확인합니다.
- 출결 정정이 필요한 경우 전자결재나 출결정정 흐름으로 이어집니다.

### 4-3. 할 일

- 개인 할 일 메모를 관리합니다.
- 간단한 업무 체크리스트 용도로 사용합니다.

### 4-4. 증명서

- 재직증명서 등 증명서 신청 내역을 확인합니다.
- 신청 상태와 발급 이력을 조회합니다.

### 4-5. 급여

- 급여명세를 확인합니다.
- 급여 항목과 지급 내역을 조회합니다.

### 4-6. 서류제출

- 인사 관련 요청 서류를 업로드합니다.
- 제출 대상 서류와 제출 현황을 확인합니다.

### 4-7. 알림

- 전자결재, 채팅, 게시판, 급여, 출퇴근 관련 알림을 확인합니다.
- 읽음 상태와 최근 알림 목록을 확인합니다.

## 5. 역할별 대시보드 차이

### 일반 직원

- 오늘 출근 상태
- 남은 연차 또는 휴가 관련 정보
- 결재 대기 건수
- 채팅 바로가기

### 부서장

- 팀 인원 현황
- 팀 승인 대기 건수
- 오늘 출근 현황
- 재고 부족 요약

### 관리자

- 결재 대기 건수
- 재고 부족 요약
- 인사관리 바로가기
- 관리자 메뉴 바로가기

## 6. 대표 사용 순서

### 급여명세 확인

1. `내정보`를 엽니다.
2. `급여` 탭을 누릅니다.
3. 지급 월 또는 최근 급여명세를 확인합니다.
4. 상세 항목이 필요하면 급여 화면 내부 정보를 확인합니다.

### 증명서 신청 상태 확인

1. `내정보`를 엽니다.
2. `증명서` 탭을 누릅니다.
3. 신청한 문서 목록을 확인합니다.
4. 발급 완료 여부와 출력 가능 상태를 확인합니다.

### 제출 서류 업로드

1. `내정보`를 엽니다.
2. `서류제출` 탭을 누릅니다.
3. 제출 대상 서류를 선택합니다.
4. 파일을 업로드합니다.
5. 제출 완료 후 목록에서 상태를 확인합니다.

### 개인 알림 확인

1. `내정보`를 엽니다.
2. `알림` 탭을 누릅니다.
3. 최근 알림을 확인합니다.
4. 필요한 경우 해당 알림을 눌러 관련 화면으로 이동합니다.

## 7. 주의사항

- 실제 보이는 카드와 메뉴는 권한과 직책에 따라 일부 달라질 수 있습니다.
- `내정보`는 개인 시작 화면이므로, 팀 전체 데이터 수정은 이 메뉴에서 하지 않습니다.
- 출결 정정이나 승인 업무는 `전자결재`, `인사관리`와 연결될 수 있습니다.
