# NEWMSO 인사관리 세부 설명서 2
## 근태 · 휴가: 근태, 교대근무, 자동편성, 연차/휴가

## 1. 문서 목적

이 문서는 `인사관리 > 근태 · 급여` 중에서도 `근태`와 `연차/휴가` 중심 기능을 설명합니다.
출퇴근 기록, 월간 현황, 근무 스케줄, 연차 사용과 촉진 기능을 단계별로 설명합니다.

## 2. 포함 메뉴

1. `근태`
2. `교대근무`
3. `근무표자동편성`
4. `연차/휴가`
5. `간호근무표`
6. `공휴일달력`

## 3. 근태

### 근태 화면 내부 탭

`근태` 화면은 별도 메뉴가 여러 개로 흩어진 구조가 아니라, 한 화면 안에서 아래 보기로 전환됩니다.

1. `대시보드`
2. `일일 현황`
3. `월간 통계`
4. `근무 일정 편성`
5. `근태 달력`

### 근태 상위 보조 탭

인사관리의 상위 보조 탭에서는 아래 분석 메뉴와 연결됩니다.

1. `근태 현황`
2. `연차소멸알림`
3. `지각조퇴분석`
4. `근무형태변경이력`
5. `조기퇴근감지`

### 대표 사용 순서

#### 일일 출결 확인

1. `인사관리 > 근태 · 급여 > 근태`로 이동합니다.
2. 보기에서 `일일 현황`을 선택합니다.
3. 날짜를 선택합니다.
4. 직원별 출근, 퇴근, 지각, 결근 상태를 확인합니다.

#### 월간 통계 확인

1. `근태` 화면을 엽니다.
2. 보기에서 `월간 통계`를 선택합니다.
3. 월을 선택합니다.
4. 출근율, 지각, 결근, 조기퇴근 통계를 확인합니다.

#### 근무 일정 편성

1. `근태` 화면을 엽니다.
2. `근무 일정 편성` 보기를 선택합니다.
3. 월을 선택합니다.
4. 직원별 날짜에 근무형태를 배정합니다.

## 4. 교대근무

### 핵심 역할

- 교대 근무표 관리
- 근무 패턴 설정
- 부서별 스케줄 운영

### 사용 목적

일반 근무표보다 교대형 스케줄 관리에 적합한 화면입니다.

## 5. 근무표자동편성

### 핵심 역할

- 자동 편성 알고리즘 기반 스케줄 생성
- 반복 수작업 감소

### 사용 목적

교대근무 화면이 수동 운영 중심이라면, 이 메뉴는 자동 배치에 더 가깝습니다.

## 6. 연차/휴가

### 연차/휴가 화면 내부 탭

1. `연차/휴가 신청내역`
2. `연차 대시보드`
3. `연차사용촉진 자동화`
4. `연차 자동부여 설정`

### 탭별 설명

#### 연차/휴가 신청내역

- 전체 휴가 신청 목록 조회
- 승인/반려 처리
- 연차 사용일수 반영

#### 연차 대시보드

- 직원별 잔여 연차와 사용 상태 확인
- 현황 요약 확인

#### 연차사용촉진 자동화

- 연차촉진 발송 관련 흐름 확인
- 자동 또는 운영형 점검에 사용

#### 연차 자동부여 설정

- 입사일 기준 또는 회계연도 기준 선택
- 연차 자동 부여 실행

### 대표 사용 순서

#### 휴가 승인 처리

1. `연차/휴가` 화면을 엽니다.
2. `연차/휴가 신청내역` 탭을 선택합니다.
3. 대기 상태 신청서를 찾습니다.
4. 내용을 확인합니다.
5. `승인` 또는 `반려` 처리합니다.

#### 연차 자동부여 실행

1. `연차/휴가` 화면을 엽니다.
2. `연차 자동부여 설정` 탭을 선택합니다.
3. `입사일 기준` 또는 `회계연도 기준`을 선택합니다.
4. 자동부여 실행 버튼을 눌러 재계산합니다.

## 7. 간호근무표

### 핵심 역할

- 간호 인력 근무표 관리
- 특화된 교대 근무 운영

### 주의사항

`교대근무`, `근무표자동편성`, `간호근무표`는 이름이 비슷하지만 목적이 다릅니다.

- `교대근무`: 일반 교대 운영
- `근무표자동편성`: 알고리즘 자동 편성
- `간호근무표`: 간호 인력 특화 근무표

## 8. 공휴일달력

### 핵심 역할

- 휴일 기준 캘린더 확인
- 스케줄 및 급여 기준 참고

## 9. 누가 주로 쓰는가

- 부서장
- 인사 담당자
- 스케줄 관리자
- 간호/운영 책임자
