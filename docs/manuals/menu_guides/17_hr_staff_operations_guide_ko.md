# NEWMSO 인사관리 세부 설명서 1
## 인력관리: 구성원현황, 인사발령, 교육, 조직도편집기

## 1. 문서 목적

이 문서는 `인사관리 > 인력관리` 영역을 별도로 설명하는 세부 안내서입니다.
직원 정보 관리, 인사 이력 관리, 교육 관리, 조직도 편집 기능을 중심으로 설명합니다.

## 2. 포함 메뉴

1. `구성원현황`
2. `인사발령`
3. `포상/징계`
4. `교육`
5. `조직도편집기`
6. `오프보딩`

## 3. 구성원현황

### 핵심 역할

- 직원 목록 조회
- 상세 인사 정보 확인
- 신규 직원 등록/수정
- 근무형태, 급여, 계약 기본정보 관리
- 인사 이력 타임라인 확인
- ESS 정보수정 요청 검토

### 실제 화면 성격

`구성원현황`은 인사관리의 기준 화면입니다.
직원 목록에서 대상을 고른 뒤 상세 정보와 변경 이력을 확인하는 구조입니다.

### 대표 사용 순서

1. `인사관리 > 인력관리 > 구성원현황`으로 이동합니다.
2. 직원 목록에서 이름, 부서, 회사 기준으로 대상을 찾습니다.
3. 상세 정보 패널 또는 모달을 엽니다.
4. 기본정보, 소속, 급여, 복지, 근무형태 정보를 확인합니다.
5. 필요 시 수정 저장합니다.

### 실무 포인트

- 신규 직원 기본값은 선택 회사 기준으로 세팅됩니다.
- 부서 목록은 조직팀 데이터와 연결됩니다.
- 근무형태와 급여 기초값은 이후 `근태`, `급여`, `계약` 화면과 연결됩니다.

## 4. 인사발령

### 핵심 역할

- 직원 발령 등록
- 부서/직책 변경 기록
- 인사 이동 이력 관리

### 사용 목적

직원의 소속, 역할, 직책이 바뀌는 시점을 공식 기록으로 남기는 기능입니다.

### 대표 사용 순서

1. `인사발령` 화면을 엽니다.
2. 대상 직원을 선택합니다.
3. 변경 전/후 부서, 직책, 발령일을 입력합니다.
4. 저장 후 구성원 정보 반영 여부를 확인합니다.

## 5. 포상/징계

### 핵심 역할

- 포상 기록 관리
- 징계 기록 관리
- 인사평가 참고 이력 누적

### 사용 목적

직원의 공식 포상 또는 징계 이력을 남기고, 향후 인사 판단의 근거 자료로 활용합니다.

## 6. 교육

### 핵심 역할

- 교육 계획 등록
- 교육 이수 확인
- 자격/교육 이력 추적

### 사용 목적

직무 교육, 필수 교육, 보수 교육 등 이수 여부를 관리하는 데 사용합니다.

### 대표 사용 순서

1. `교육` 화면을 엽니다.
2. 대상자 또는 대상 부서를 선택합니다.
3. 교육명, 일정, 이수 여부를 등록합니다.
4. 완료 후 이수 현황을 갱신합니다.

## 7. 조직도편집기

### 핵심 역할

- 조직도 조회가 아니라 편집
- 조직 구조 수정
- 부서/팀 배치 조정

### 주의사항

- `추가기능 > 조직도` 또는 별도 조직도 화면은 조회용에 가깝습니다.
- `조직도편집기`는 인사 담당자가 실제 구조를 수정하는 편집 메뉴입니다.

## 8. 오프보딩

### 핵심 역할

- 퇴사 체크리스트 관리
- 종료 절차 확인
- 인수인계 상태 점검

### 대표 사용 순서

1. 퇴사 예정 직원을 선택합니다.
2. 반납, 권한 회수, 인수인계 상태를 점검합니다.
3. 관련 문서와 종료 일정을 확인합니다.
4. 체크리스트를 완료 처리합니다.

## 10. 누가 주로 쓰는가

- 인사 담당자
- 부서장
- 관리자

일반 직원에게는 대부분 읽기 또는 비노출이 적합한 영역입니다.
