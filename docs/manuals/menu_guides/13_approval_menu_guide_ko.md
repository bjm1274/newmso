# NEWMSO 전자결재 상세 설명서

## 1. 메뉴 개요

`전자결재`는 기안 작성, 승인, 결재 이력 확인, 양식 관리까지 처리하는 문서 승인 메뉴입니다.
현재 원본 UI에서는 일반 사용자 화면에서도 관리성 탭이 일부 보일 수 있으므로, 실제 사용자는 권한에 맞는 탭만 사용하도록 안내해야 합니다.

## 2. 진입 방법

1. 좌측 사이드바에서 `전자결재`를 누릅니다.
2. 사이드바 빠른 메뉴 또는 화면 상단 탭에서 필요한 화면으로 이동합니다.

## 3. 빠른 메뉴와 전체 탭 구조

### 사이드바 빠른 메뉴

1. `기안함`
2. `결재함`
3. `작성하기`
4. `캘린더`

### 전자결재 화면 내부 전체 탭

1. `기안함`
2. `결재함`
3. `작성하기`
4. `캘린더`
5. `양식빌더`
6. `서명관리`
7. `직인관리`

## 4. 탭별 기능 설명

### 기안함

- 내가 작성한 문서를 확인합니다.
- 상태별로 진행 중, 승인 완료, 반려 문서를 확인합니다.

### 결재함

- 내가 처리해야 할 승인 문서를 확인합니다.
- 일괄 승인, 일괄 반려 기능이 연결됩니다.

### 작성하기

- 새 결재 문서를 작성합니다.
- 결재선, 참조선, 제목, 내용, 첨부 정보를 입력합니다.
- 임시저장, 자동저장, 이전 기안 불러오기 기능이 있습니다.

### 캘린더

- 결재 일정이나 관련 일정을 달력 형태로 확인합니다.

### 양식빌더

- 결재 양식을 설계하거나 관리하는 용도의 화면입니다.
- 현재 원본 UI에서는 일반 사용자에게도 보일 수 있으나 실제 사용 대상은 제한적입니다.

### 서명관리

- 개인 서명 또는 결재 서명을 관리합니다.

### 직인관리

- 직인 또는 승인용 도장 이미지를 관리합니다.

## 5. 작성하기에서 선택 가능한 문서 유형

현재 기본 문서 유형은 아래와 같습니다.

1. `인사명령`
2. `연차/휴가`
3. `연차계획서`
4. `연장근무`
5. `물품신청`
6. `수리요청서`
7. `업무기안`
8. `업무협조`
9. `양식신청`
10. `출결정정`

## 6. 대표 사용 순서

### 연차/휴가 결재 올리기

1. `전자결재`를 엽니다.
2. `작성하기`를 누릅니다.
3. 문서 유형에서 `연차/휴가`를 선택합니다.
4. 제목과 상세 내용을 입력합니다.
5. 결재선과 참조선을 지정합니다.
6. 필요하면 이전 기안 불러오기 또는 임시저장 기능을 사용합니다.
7. 제출 후 `기안함`에서 상태를 확인합니다.

### 결재 승인 처리

1. `전자결재`를 엽니다.
2. `결재함`으로 이동합니다.
3. 대기 문서를 선택합니다.
4. 내용을 검토합니다.
5. `승인` 또는 `반려`를 처리합니다.
6. 다건 처리 시 일괄 승인/반려 기능을 사용합니다.

### 서명 등록

1. `전자결재`를 엽니다.
2. `서명관리`를 선택합니다.
3. 서명 이미지를 등록하거나 교체합니다.
4. 저장 후 문서 작성 화면에서 적용 여부를 확인합니다.

## 7. 주의사항

- 전자결재는 공식 기록이 남는 승인 도구입니다.
- 채팅에서 논의한 내용은 전자결재로 최종 기록을 남기는 방식이 안전합니다.
- `양식빌더`, `서명관리`, `직인관리`는 실제 운영상 일반 직원보다 관리자/승인자 사용 비중이 높습니다.
