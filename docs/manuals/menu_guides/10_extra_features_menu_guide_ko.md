# NEWMSO 추가기능 상세 설명서

## 1. 메뉴 개요

`추가기능`은 고정 서브메뉴형 화면이 아니라, 카드형 바로가기 화면입니다.
공통 보조 기능과 병원 운영용 기능, 외부 링크가 함께 모여 있습니다.

## 2. 진입 방법

1. 좌측 사이드바에서 `추가기능`을 누릅니다.
2. 화면에 보이는 카드 또는 링크를 선택합니다.

## 3. 카드 구조

현재 원본 기준 카드는 아래와 같습니다.

1. `조직도`
2. `부서별 재고`
3. `근무현황`
4. `인계노트`
5. `퇴원심사`
6. `마감보고`
7. `직원평가`

외부 링크:

- `KM Park`
- `U+ 웹팩스`

## 4. 카드별 기능 설명

### 조직도

- 조직도 화면으로 이동합니다.
- 직원 검색, 연락처 확인, 조직 구조 확인 용도로 사용합니다.

### 부서별 재고

- 부서 단위 재고 상태를 빠르게 확인합니다.
- 재고관리 전체 화면으로 들어가기 전 요약 확인용으로 사용합니다.

### 근무현황

- 현재 근무 상태를 빠르게 확인합니다.
- 출근 현황, 부서별 근무 상황 파악에 사용합니다.

### 인계노트

- 인수인계 메모 또는 교대 메모 용도로 사용합니다.
- 의료기관 운영 환경에서 실무용으로 자주 사용될 수 있습니다.

### 퇴원심사

- 퇴원 관련 점검, 검토 흐름에 사용합니다.
- 부서나 역할에 따라 보임 여부가 달라질 수 있습니다.

### 마감보고

- 일일 마감 보고를 입력하거나 확인합니다.
- 운영 마감 업무와 연결됩니다.

### 직원평가

- 직원 평가 기록 또는 점검 업무에 사용합니다.
- 관리자/부서장 권한 환경에서 중요도가 높습니다.

## 5. 대표 사용 순서

### 조직도 바로가기 사용

1. `추가기능`을 엽니다.
2. `조직도` 카드를 누릅니다.
3. 직원 이름 또는 부서를 검색합니다.
4. 연락처와 소속 정보를 확인합니다.

### 부서별 재고 빠른 확인

1. `추가기능`을 엽니다.
2. `부서별 재고` 카드를 누릅니다.
3. 필요한 부서를 선택합니다.
4. 현재 부족 품목이나 핵심 재고 상태를 확인합니다.

### 외부 링크 사용

1. `추가기능`을 엽니다.
2. 하단 또는 링크 영역에서 `KM Park` 또는 `U+ 웹팩스`를 선택합니다.
3. 외부 시스템으로 이동해 해당 업무를 처리합니다.

## 6. 주의사항

- `추가기능`은 메뉴 허브 성격이 강하므로, 세부 편집은 연결된 기능 화면에서 처리합니다.
- 의료기관 특화 기능은 일반 직원에게 모두 필요한 메뉴가 아닐 수 있습니다.
- 일부 카드는 역할이나 권한에 따라 숨겨질 수 있습니다.
