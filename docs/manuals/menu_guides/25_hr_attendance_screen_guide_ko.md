# NEWMSO 인사관리 화면별 설명서 2
## 근태 화면 상세 설명서

## 1. 화면 목적

`근태` 화면은 출퇴근 기록과 월간 현황, 근무표 편성, 통계, 달력 조회를 한 곳에서 처리하는 통합 화면입니다.
현재 원본 기준으로 별도 메뉴가 여러 개로 분리된 구조가 아니라, 하나의 화면 안에서 보기 모드를 바꾸는 방식입니다.

## 2. 어디서 들어가는가

1. 좌측 사이드바에서 `인사관리`를 누릅니다.
2. `근태 · 급여` 업무 공간을 선택합니다.
3. `근태` 탭을 선택합니다.

## 3. 근태 화면 보기 모드

현재 원본 기준 보기 모드는 아래 5가지입니다.

1. `대시보드`
2. `일별 현황`
3. `월별 대장`
4. `근무표 편성`
5. `근태 달력`

## 4. 보기별 설명

### 대시보드

- 출근율
- 지각/조퇴/결근 요약
- 위험 직원 추적
- 월간 통계 요약

### 일별 현황

- 특정 날짜 기준 직원별 출퇴근 상태 확인
- 출근, 퇴근, 근무시간, 비고 확인

### 월별 대장

- 월 단위 출결 대장 조회
- 직원별 전체 일자 확인

### 근무표 편성

- 월별 근무 스케줄 배정
- `work_shifts` 기반 근무형태 조회
- `shift_assignments` 저장

### 근태 달력

- 달력 형태로 월간 출결 확인

## 5. 화면 상단 도구

### 날짜/월 선택

- 일별 현황에서는 날짜 선택
- 나머지 보기에서는 월 선택

### 상태 일괄 수정

- 벌크 수정 모달을 열어 여러 직원 또는 여러 날짜의 상태를 한 번에 수정
- 일/주/월/사용자 범위 선택 가능

## 6. 근태와 연결되는 상위 보조 탭

인사관리 상위 보조 흐름에서는 아래 분석 메뉴와 이어집니다.

1. `근태 현황`
2. `연차소멸알림`
3. `지각조퇴분석`
4. `근무형태변경이력`
5. `조기퇴근감지`

## 7. 대표 사용 순서

### 오늘 출결 확인

1. `근태` 화면을 엽니다.
2. `일별 현황`을 선택합니다.
3. 날짜를 오늘로 둡니다.
4. 직원별 출근 상태를 확인합니다.

### 월간 출결 점검

1. `근태` 화면을 엽니다.
2. `월별 대장`을 선택합니다.
3. 확인할 월을 선택합니다.
4. 직원별 누락, 지각, 결근 여부를 점검합니다.

### 근무표 배정

1. `근태` 화면을 엽니다.
2. `근무표 편성`을 선택합니다.
3. 월을 선택합니다.
4. 직원별 날짜에 근무형태를 지정합니다.
5. 저장된 스케줄을 다시 확인합니다.

### 상태 일괄 수정

1. `근태` 화면을 엽니다.
2. 상단의 `상태 일괄 수정` 버튼을 누릅니다.
3. 적용 범위를 선택합니다.
4. 시작일/종료일을 지정합니다.
5. 상태를 선택한 뒤 저장합니다.

## 8. 실무 포인트

- 근태 데이터는 `attendances`를 기준으로 조회됩니다.
- 근무표 편성은 `shift_assignments`에 직접 연결됩니다.
- 근무형태가 정확하지 않으면 월별 통계와 급여 연동에 오류가 생길 수 있습니다.
