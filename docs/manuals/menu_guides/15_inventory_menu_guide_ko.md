# NEWMSO 재고관리 상세 설명서

## 1. 메뉴 개요

`재고관리`는 재고 현황, 입출고, 발주, 자산, 점검, 통계까지 함께 다루는 통합 메뉴입니다.
현재 원본 UI에서는 17개 서브메뉴가 평면적으로 나열되어 있습니다.

## 2. 진입 방법

1. 좌측 사이드바에서 `재고관리`를 누릅니다.
2. 플라이아웃 또는 내부 메뉴에서 원하는 서브메뉴를 선택합니다.

## 3. 서브메뉴 전체 목록

1. `현황`
2. `이력`
3. `UDI`
4. `명세서`
5. `발주`
6. `스캔`
7. `등록`
8. `자산 QR`
9. `AS / 반품`
10. `거래처`
11. `재고 실사`
12. `유통기한 알림`
13. `재고 이관`
14. `카테고리`
15. `소모품 통계`
16. `납품 확인서`
17. `수요 예측`

## 4. 서브메뉴별 기능 설명

### 현황

- 현재 재고 수량과 상태를 조회합니다.
- 부족 품목 확인의 시작 화면 역할을 합니다.

### 이력

- 입출고와 변경 이력을 확인합니다.

### UDI

- UDI 기준 식별 정보를 관리합니다.

### 명세서

- 재고 관련 명세서를 확인합니다.

### 발주

- 구매 및 발주 요청을 관리합니다.

### 스캔

- 바코드/QR 기반 재고 처리에 사용합니다.

### 등록

- 신규 품목 등록 화면입니다.
- 내부 등록 방식은 `form`, `excel`, `auto_extract` 3가지가 있습니다.

### 자산 QR

- 자산용 QR 관리 기능입니다.

### AS / 반품

- AS 요청과 반품 상태를 관리합니다.

### 거래처

- 공급업체 및 거래처 정보를 관리합니다.

### 재고 실사

- 실사 결과와 실제 수량 점검을 처리합니다.

### 유통기한 알림

- 유통기한 임박 재고를 확인합니다.

### 재고 이관

- 부서 간 재고 이동을 처리합니다.

### 카테고리

- 재고 분류 체계를 관리합니다.

### 소모품 통계

- 사용량과 소비 패턴을 분석합니다.

### 납품 확인서

- 납품 관련 문서를 확인합니다.

### 수요 예측

- 향후 사용량 예측 기능을 제공합니다.

## 5. 대표 사용 순서

### 부족 재고 확인

1. `재고관리`를 엽니다.
2. `현황`을 선택합니다.
3. 부족 품목 또는 임계 수량 항목을 확인합니다.
4. 필요하면 `발주`로 이동해 구매 요청을 진행합니다.

### 신규 품목 등록

1. `재고관리`를 엽니다.
2. `등록`을 선택합니다.
3. 등록 방식을 선택합니다.
4. 직접 입력, 엑셀 업로드, 자동 추출 중 한 가지 방식으로 등록합니다.
5. 저장 후 `현황`에서 반영 여부를 확인합니다.

### 유통기한 임박 품목 확인

1. `재고관리`를 엽니다.
2. `유통기한 알림`을 선택합니다.
3. 임박 항목을 확인합니다.
4. 필요하면 폐기, 사용 우선, 이관, 발주 조정 업무로 연결합니다.

## 6. 운영 포인트

- 부족 재고 알림과 유통기한 알림이 연결됩니다.
- 저재고 상황에서 자동 승인 요청 흐름이 붙는 기능이 있습니다.
- 등록과 발주, 거래처, 통계가 서로 연결되므로 데이터 정확도가 중요합니다.

## 7. 주의사항

- 재고 실사와 현황 수치가 다르면 먼저 이력과 실사 결과를 확인해야 합니다.
- `등록`은 가장 영향이 큰 메뉴 중 하나이므로 잘못된 단위나 수량 입력을 주의해야 합니다.
- 의료기관 환경에서는 유통기한과 자산 QR 관리 중요도가 높습니다.
