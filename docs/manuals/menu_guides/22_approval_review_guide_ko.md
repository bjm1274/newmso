# NEWMSO 전자결재 세부 설명서 2
## 기안함, 결재함, 캘린더

## 1. 문서 목적

이 문서는 `전자결재`의 조회와 승인 처리 기능을 설명합니다.
직원은 `기안함`, 승인자는 `결재함`, 일정 확인자는 `캘린더`를 주로 사용합니다.

## 2. 포함 탭

1. `기안함`
2. `결재함`
3. `캘린더`

## 3. 기안함

### 핵심 역할

- 내가 올린 문서 목록 확인
- 진행 상태 확인
- 승인 완료 / 반려 문서 추적

### 확인할 항목

- 문서 제목
- 문서 유형
- 작성일
- 현재 상태
- 현재 결재자

### 언제 쓰는가

- 문서를 올린 뒤 승인 진행 상황을 볼 때
- 반려 사유를 확인할 때
- 이전에 올린 문서를 다시 참고할 때

## 4. 결재함

### 핵심 역할

- 내가 처리해야 할 결재 문서 확인
- 승인/반려 처리
- 다건 일괄 승인/반려

### 주요 기능

- 상태 필터
- 선택 항목 일괄 승인
- 선택 항목 일괄 반려
- 반려 사유 저장

### 대표 사용 순서

1. `전자결재 > 결재함`을 엽니다.
2. 대기 문서를 선택합니다.
3. 내용과 첨부를 검토합니다.
4. `승인` 또는 `반려`를 선택합니다.
5. 여러 건이면 일괄 처리 기능을 사용합니다.

### 주의사항

- 일괄 승인은 빠르지만, 문서 내용을 생략하고 처리하면 안 됩니다.
- 반려 시 사유를 남겨야 작성자가 수정 방향을 알 수 있습니다.

## 5. 캘린더

### 핵심 역할

- 결재와 관련된 일정 또는 문서 일정을 달력 형태로 확인

### 사용 목적

- 특정 기간에 몰린 결재 문서를 파악
- 휴가/업무 문서와 일정성을 함께 확인

## 6. 대표 사용 순서

### 내가 올린 문서 상태 확인

1. `기안함`으로 이동합니다.
2. 최근 문서를 찾습니다.
3. 현재 승인 단계 또는 반려 상태를 확인합니다.

### 부서장이 승인 처리하기

1. `결재함`으로 이동합니다.
2. 본인이 처리할 문서를 선택합니다.
3. 결재 내용을 검토합니다.
4. 승인 또는 반려를 처리합니다.
5. 완료 후 남은 대기 건수를 다시 확인합니다.

## 7. 누가 주로 쓰는가

- 일반 직원: `기안함`
- 팀장/부서장/승인권자: `결재함`
- 관리자/운영자: `캘린더` 포함 전체
