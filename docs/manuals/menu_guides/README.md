## 공통 읽는 법

- `메인 메뉴`: 좌측 고정 사이드바에서 누르는 1차 메뉴
- `서브메뉴`: 사이드바 플라이아웃 또는 화면 상단에서 고르는 2차 메뉴
- `내부 탭`: 특정 화면 안에서 한 번 더 나뉘는 보조 탭
- `대표 사용 순서`: 실제 업무에서 가장 많이 쓰는 흐름을 단계별로 정리한 부분

## 배포 권장 방식

- 일반 사원: `내정보`, `추가기능`, `채팅`, `게시판`, `전자결재`
- 부서장: 위 문서 + `인사관리`, `재고관리`
- 관리자/MSO: 위 문서 + `관리자`
