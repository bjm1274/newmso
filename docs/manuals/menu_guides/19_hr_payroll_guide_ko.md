# NEWMSO 인사관리 세부 설명서 3
## 급여: 급여관리, 정산, 4대보험, 시뮬레이션

## 1. 문서 목적

이 문서는 `인사관리 > 근태 · 급여 > 급여` 화면을 별도로 설명합니다.
현재 원본 기준 급여 화면은 단순 급여명세 확인이 아니라, 대시보드, 정산, 퇴직금, 4대보험, 이상 탐지까지 포함한 대형 메뉴입니다.

## 2. 급여 화면 내부 탭

현재 원본 기준 내부 탭은 아래와 같습니다.

1. `대시보드`
2. `급여정산`
3. `급여대장`
4. `연말/퇴직정산`
5. `통합설정`
6. `급여 시뮬레이터`
7. `4대보험 EDI`
8. `퇴직연금`
9. `임금피크제`
10. `최저임금 체크`
11. `통상임금 계산기`
12. `비과세 한도 체크`
13. `총인건비 예측`
14. `세전/세후 비교`
15. `미지급 수당 알림`
16. `급여 고도화`
17. `무급 결근 차감`

## 3. 주요 탭 설명

### 대시보드

- 인사/급여 통합 대시보드
- 주52시간 모니터링
- 이직률, 연차, 급여 분포, 인건비 추이 등 요약 확인

### 급여정산

- 월별 급여 정산 실행
- 정산 프로세스 시작 화면

### 급여대장

- 직원별 급여명세 확인
- 급여대장 목록 확인
- 급여 변동 이력 조회

### 연말/퇴직정산

- 연말정산 처리
- 퇴직금 계산
- 퇴직/연차 연계 대시보드 확인

### 통합설정

- 급여 운영에 필요한 통합 설정값 관리

### 급여 시뮬레이터

- 기본급과 수당을 바탕으로 실시간 계산

### 4대보험 EDI

- 4대보험 관련 처리와 EDI 연동 보조 기능

### 퇴직연금

- 퇴직연금 관리

### 임금피크제

- 임금피크제 계산 및 검토

### 최저임금 체크 / 통상임금 계산기 / 비과세 한도 체크

- 법정 기준 점검용 도구

### 총인건비 예측 / 세전세후 비교 / 미지급 수당 알림 / 무급 결근 차감

- 급여 운영 분석과 이상 징후 확인용 도구

### 급여 고도화

- 고급 급여 운영 기능 묶음
- 추가 계산, 공제, 운영 보조 기능 포함

## 4. 대표 사용 순서

### 월 급여 정산

1. `인사관리 > 근태 · 급여 > 급여`로 이동합니다.
2. `급여정산` 탭을 선택합니다.
3. 대상 월을 선택합니다.
4. 정산 실행 후 결과를 확인합니다.
5. 필요 시 `급여대장`에서 직원별 결과를 검토합니다.

### 직원별 급여명세 확인

1. `급여대장` 탭을 엽니다.
2. 직원을 선택합니다.
3. 해당 월 기록을 확인합니다.
4. 상세 항목과 급여 변동 이력을 확인합니다.

### 법정 기준 점검

1. 급여 화면을 엽니다.
2. `최저임금 체크`, `통상임금 계산기`, `비과세 한도 체크` 중 필요한 탭을 선택합니다.
3. 기준값을 입력하거나 대상을 선택합니다.
4. 위반 여부 또는 계산 결과를 확인합니다.

### 4대보험 처리

1. 급여 화면을 엽니다.
2. `4대보험 EDI` 탭을 선택합니다.
3. 대상 회사와 직원을 확인합니다.
4. 필요한 보험 처리나 파일 점검을 진행합니다.

## 5. 대시보드 탭에서 보는 항목

현재 통합 대시보드에는 아래 성격의 카드가 연결됩니다.

- 급여 내보내기
- 주52시간 모니터링
- 이직률/근속률
- 퇴직금/연차 대시보드
- 연차 종합 대시보드
- 부서별 급여분포
- 인건비 추이

## 6. 실무 포인트

- 급여 화면은 `급여 확인` 수준이 아니라 운영자가 쓰는 통합 센터에 가깝습니다.
- 연말정산, 퇴직금, 4대보험, 법정기준 체크가 모두 이 화면 안에 모여 있습니다.
- 따라서 일반 직원 안내서에는 이 전체를 그대로 노출하기보다, 운영자용/담당자용으로 분리 설명하는 것이 좋습니다.
