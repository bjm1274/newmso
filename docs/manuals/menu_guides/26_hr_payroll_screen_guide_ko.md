# NEWMSO 인사관리 화면별 설명서 3
## 급여 화면 상세 설명서

## 1. 화면 목적

`급여` 화면은 급여 정산, 급여대장 조회, 연말정산, 퇴직금, 4대보험, 각종 계산기와 분석 도구까지 포함한 통합 운영 화면입니다.
일반 직원 조회용이 아니라 운영자 중심의 센터에 가깝습니다.

## 2. 어디서 들어가는가

1. 좌측 사이드바에서 `인사관리`를 누릅니다.
2. `근태 · 급여` 업무 공간을 선택합니다.
3. `급여` 탭을 선택합니다.

## 3. 급여 화면 내부 탭

현재 원본 기준 탭은 아래와 같습니다.

1. `대시보드`
2. `급여정산`
3. `급여대장`
4. `연말퇴직정산`
5. `통합설정`
6. `급여 시뮬레이터`
7. `4대보험 EDI`
8. `퇴직연금`
9. `임금피크제`
10. `최저임금 체크`
11. `통상임금 계산기`
12. `비과세 한도 체크`
13. `총인건비 예측`
14. `세전/세후 비교`
15. `미지급 수당 알림`
16. `급여 고도화`
17. `무급 결근 차감`

## 4. 핵심 탭 설명

### 대시보드

- 인사/급여 통합 대시보드
- 주52시간 모니터링
- 이직률, 연차, 급여분포, 인건비 추이 확인

### 급여정산

- 월 급여 정산 실행
- 결근/지각 반영
- 4대보험 재계산 연동

### 급여대장

- 직원별 급여 기록 확인
- 급여 명세 상세 확인
- 급여 변동 이력 확인

### 연말퇴직정산

- 연말정산 처리
- 퇴직금 계산
- 퇴직/연차 연계 대시보드 확인

### 통합설정

- 급여 운영 공통 설정 관리

### 급여 시뮬레이터

- 기본급과 수당으로 실시간 계산

### 4대보험 EDI

- 4대보험 관련 처리 지원

### 급여 고도화

- 추가 계산과 공제 보조 기능 모음

## 5. 대표 사용 순서

### 월 급여 정산

1. `급여` 화면을 엽니다.
2. `급여정산` 탭을 선택합니다.
3. 기준 월을 선택합니다.
4. 정산을 실행합니다.
5. `급여대장`에서 결과를 확인합니다.

### 직원별 급여 확인

1. `급여대장` 탭을 엽니다.
2. 직원을 선택합니다.
3. 해당 월 기록을 선택합니다.
4. 지급 항목과 공제 항목을 검토합니다.

### 퇴직 정산 확인

1. `연말퇴직정산` 탭을 엽니다.
2. 대상 직원을 선택합니다.
3. 연말정산 또는 퇴직금 계산을 진행합니다.
4. 결과를 검토합니다.

### 법정 기준 체크

1. `최저임금 체크`, `통상임금 계산기`, `비과세 한도 체크` 중 필요한 탭을 엽니다.
2. 기준값을 입력합니다.
3. 계산 결과를 확인합니다.

## 6. 월 선택과 회사 기준

- 상단에서 월을 선택할 수 있습니다.
- 회사 기준으로 필터링된 직원 목록을 대상으로 처리합니다.

## 7. 실무 포인트

- `급여정산`과 `급여대장`은 함께 움직입니다.
- 급여 정산 전에 직원 기본급, 수당, 보험 설정이 `구성원현황`에서 올바르게 입력되어 있어야 합니다.
- 법정 기준 체크 결과는 운영 참고용이므로 최종 지급 전 재검토가 필요합니다.
