# NEWMSO 관리자 상세 설명서

## 1. 메뉴 개요

`관리자`는 경영분석, 감사, 권한, 회사 운영 설정, 백업/초기화까지 포함하는 최고 수준 운영 메뉴입니다.
현재 원본 코드 기준 실제 화면은 MSO 권한 계정 중심으로 동작합니다.

## 2. 진입 방법

1. 좌측 사이드바에서 `관리자`를 누릅니다.
2. 상단 탭 또는 내부 하위 탭에서 필요한 관리 기능을 선택합니다.

## 3. 상위 탭 구조

현재 원본 기준 상위 탭은 아래와 같습니다.

1. `경영분석`
2. `감사센터`
3. `엑셀등록`
4. `알림자동화`
5. `연차수동부여`
6. `회사관리`
7. `직원권한`
8. `수술검사템플릿`
9. `팝업관리`
10. `데이터백업`
11. `데이터초기화`
12. `양식빌더`
13. `문서서식`
14. `급여이상치`
15. `공문서대장`

## 4. 내부 하위 탭 구조

### 경영분석 내부 탭

1. `경영대시보드`
2. `재무대시보드`
3. `예산관리`
4. `통합보고서`
5. `법인손익`

### 감사센터 내부 탭

1. `접근감사로그`
2. `감사로그`

## 5. 상위 탭별 기능 설명

### 경영분석

- 경영 KPI, 재무, 예산, 보고서를 확인합니다.

### 감사센터

- 사용자 접근 기록과 감사 로그를 조회합니다.

### 엑셀등록

- 데이터 일괄 등록에 사용합니다.

### 알림자동화

- 자동 알림 스케줄과 발송 규칙을 관리합니다.

### 연차수동부여

- 연차를 수동 지급 또는 조정합니다.

### 회사관리

- 회사 기본 운영 설정과 일부 관리 항목을 다룹니다.

### 직원권한

- 메인 메뉴 권한, 인사 권한, 관리자 권한을 조정합니다.

### 수술검사템플릿

- 병원 운영용 템플릿을 관리합니다.

### 팝업관리

- 홈페이지 또는 외부 소비처용 팝업 데이터를 관리합니다.

### 데이터백업

- 백업과 복원 관련 작업을 수행합니다.

### 데이터초기화

- 데이터 초기화 기능입니다.
- 프로덕션 운영에서는 매우 신중해야 합니다.

### 양식빌더

- 관리자용 양식 설계 기능입니다.

### 문서서식

- 문서 템플릿과 서식을 관리합니다.

### 급여이상치

- 급여 데이터 이상 탐지에 사용합니다.

### 공문서대장

- 공문서 발송과 관리 대장 기능입니다.

## 6. 대표 사용 순서

### 직원 권한 수정

1. `관리자`를 엽니다.
2. `직원권한`을 선택합니다.
3. 직원 목록에서 대상을 찾습니다.
4. 메인 메뉴 권한, 인사 권한, 관리자 권한을 조정합니다.
5. 저장 후 실제 메뉴 노출 상태를 다시 확인합니다.

### 경영 대시보드 확인

1. `관리자`를 엽니다.
2. `경영분석`을 선택합니다.
3. 내부 탭에서 `경영대시보드` 또는 `재무대시보드`를 선택합니다.
4. KPI와 보고 수치를 검토합니다.

### 접근 로그 확인

1. `관리자`를 엽니다.
2. `감사센터`를 선택합니다.
3. 내부 탭에서 `접근감사로그` 또는 `감사로그`를 선택합니다.
4. 사용자, 날짜, 이벤트 기준으로 조회합니다.

## 7. 고위험 기능 주의사항

### 데이터초기화

- 실수로 실행하면 운영 데이터에 큰 영향을 줄 수 있습니다.
- 운영 환경에서는 별도 승인과 확인 절차가 필요합니다.

### 데이터백업

- 복원 전에 백업본 기준 시점을 반드시 확인해야 합니다.

### 직원권한

- 권한 변경 직후 실제 메뉴 노출과 화면 접근 여부를 점검해야 합니다.

## 8. 운영 원칙

- 관리자 메뉴는 일반 직원 안내서에 그대로 배포하지 않는 것이 좋습니다.
- 고위험 메뉴는 관리자 교육용 문서와 함께 별도 승인 절차를 같이 설명해야 합니다.
- `관리자`는 단독 메뉴가 아니라 `인사관리`, `전자결재`, `재고관리` 운영 흐름과 연결해서 이해해야 합니다.
