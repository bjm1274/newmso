# NEWMSO 인사관리 화면별 설명서 1
## 구성원현황 화면 상세 설명서

## 1. 화면 목적

`구성원현황`은 인사관리에서 가장 기본이 되는 직원 관리 화면입니다.
직원 목록 조회, 신규 등록, 정보 수정, 근무형태 관리, 급여/보험 기초값 관리, ESS 변경 요청 검토가 이 화면에 모여 있습니다.

## 2. 어디서 들어가는가

1. 좌측 사이드바에서 `인사관리`를 누릅니다.
2. `인력관리` 업무 공간을 선택합니다.
3. `구성원현황` 탭을 선택합니다.

## 3. 화면 구성

### 좌측 또는 메인 목록 영역

- 직원 목록
- 회사/상태 기준 필터
- 사번, 성명/직함, 소속, 부서/팀, 입사일, 근무형태, 상태 확인

### 상세 관리 영역

- 신규 직원 등록 모달
- 기존 직원 수정 모달
- 인사 이력 타임라인
- 온보딩/체크리스트 연계

### 관리 보조 기능

- ESS 정보변경 승인 대기함
- 근무형태 목록 로드
- 팀 목록 캐시

## 4. 신규 등록/수정 탭 구조

직원 등록 및 수정 모달은 현재 원본 기준으로 아래 3개 탭으로 나뉩니다.

1. `기본`
2. `소속`
3. `급여`

실제 탭 라벨은 아래 의미로 사용됩니다.

- `기본`: 인적사항
- `소속`: 소속/근무
- `급여`: 급여/보험

## 5. 탭별 입력 내용

### 기본

- 이름
- 연락처
- 이메일
- 주소
- 주민번호
- 입사일 / 퇴사일
- 면허/자격 기초 정보
- 상태

### 소속

- 회사
- 부서/팀
- 직함
- 근무형태
- 고용형태
- 계약 종료일
- 주당 근무시간/근무일수

### 급여

- 기본급
- 식대
- 차량지원
- 보육수당
- 직책수당
- 연구수당
- 기타 비과세
- 사회보험 체크
- 복지 상태

## 6. ESS 변경 요청 검토

이 화면에는 `ESS(직원 셀프 서비스) 변경 요청` 승인 기능이 있습니다.

### 할 수 있는 일

- 직원이 내정보에서 수정 요청한 내용 검토
- 승인
- 반려

### 확인 가능한 변경 예시

- 주소
- 은행
- 계좌번호
- 연락처

### 처리 순서

1. `ESS 승인 대기` 표시를 확인합니다.
2. 모달을 열어 요청 건을 확인합니다.
3. 변경 전/후 내용을 검토합니다.
4. `승인` 또는 `반려`를 선택합니다.

## 7. 대표 사용 순서

### 신규 직원 등록

1. `구성원현황`을 엽니다.
2. 신규 등록 버튼을 누릅니다.
3. `기본` 탭에서 인적사항을 입력합니다.
4. `소속` 탭에서 회사, 부서, 근무형태를 입력합니다.
5. `급여` 탭에서 급여/보험 기초값을 입력합니다.
6. 저장 후 목록 반영 여부를 확인합니다.

### 직원 정보 수정

1. 목록에서 직원을 선택합니다.
2. 수정 모달을 엽니다.
3. 필요한 탭으로 이동합니다.
4. 변경사항을 저장합니다.
5. 관련 화면에서 반영 상태를 확인합니다.

### ESS 요청 승인

1. ESS 승인 대기 건수를 확인합니다.
2. 요청 상세를 엽니다.
3. 수정 항목을 비교합니다.
4. 승인 또는 반려를 처리합니다.

## 8. 실무 포인트

- 이 화면에서 설정한 기본급, 보험 여부, 근무형태는 `급여`, `근태`, `계약` 화면과 연결됩니다.
- 잘못 입력된 입사일은 연차 계산과 근속 계산에 영향을 줍니다.
- 근무형태가 비어 있으면 근태 편성이나 교대 운영에 문제가 생길 수 있습니다.
