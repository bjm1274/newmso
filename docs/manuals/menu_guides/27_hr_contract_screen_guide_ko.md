# NEWMSO 인사관리 화면별 설명서 4
## 계약 화면 상세 설명서

## 1. 화면 목적

`계약` 화면은 직원 계약 현황 조회와 신규/변경 계약 발송, 연봉 갱신 계약 발송을 처리하는 전자 계약 운영 화면입니다.
좌측에서 대상 직원을 고르고, 우측에서 계약서를 미리보는 구조입니다.

## 2. 어디서 들어가는가

1. 좌측 사이드바에서 `인사관리`를 누릅니다.
2. `복지 · 문서` 업무 공간을 선택합니다.
3. `계약` 탭을 선택합니다.

## 3. 계약 화면 상단 탭

현재 원본 기준 탭은 아래와 같습니다.

1. `계약현황`
2. `신규/변경계약서`
3. `연봉계약갱신`

## 4. 화면 구조

### 좌측 영역

- 계약 대상 직원 목록
- 선택 체크박스
- 계약 발송 대상 선택
- 신규/변경/갱신용 입력 패널

### 우측 영역

- 계약서 미리보기
- 선택 직원 계약서 내용 확인

## 5. 탭별 기능 설명

### 계약현황

- 직원별 계약 상태 조회
- 서명 대기/완료 상태 확인
- 발송 이력 확인

### 신규/변경계약서

- `신규` 또는 `변경` 계약 유형 선택
- 기본급, 비과세, 근무시간 등 조건 입력
- 다건 선택 후 계약서 발송

### 연봉계약갱신

- 연봉 갱신 계약 발송
- 적용일과 갱신 조건 입력

## 6. 발송 시 실제 처리되는 내용

현재 원본 기준 계약 발송 시 아래 흐름이 연결됩니다.

- `employment_contracts` 테이블 업서트
- 직원 급여/비과세 값 갱신
- 대상 직원에게 알림 발송

즉, 계약 화면은 문서 보기만 하는 화면이 아니라 실제 인사/급여 데이터에 영향을 주는 운영 화면입니다.

## 7. 대표 사용 순서

### 신규 계약서 발송

1. `계약` 화면을 엽니다.
2. `신규/변경계약서` 탭을 선택합니다.
3. `신규` 유형을 선택합니다.
4. 대상 직원 체크박스를 선택합니다.
5. 기본급과 근무조건을 입력합니다.
6. 발송 버튼을 누릅니다.

### 변경 계약서 발송

1. `신규/변경계약서` 탭을 엽니다.
2. `변경` 유형을 선택합니다.
3. 변경 대상 직원을 선택합니다.
4. 변경할 급여/근무조건을 입력합니다.
5. 계약서를 발송합니다.

### 연봉 갱신 계약 발송

1. `연봉계약갱신` 탭을 엽니다.
2. 대상 직원을 선택합니다.
3. 갱신 기본급과 적용일을 입력합니다.
4. 발송합니다.

## 8. 주의사항

- 계약 발송은 단순 문서 전송이 아니라 직원 데이터와 알림까지 함께 갱신합니다.
- 잘못된 기본급 입력은 이후 `급여` 화면 계산에 직접 영향을 줍니다.
- 다건 선택 발송 시 대상자를 다시 한 번 확인해야 합니다.
