# NEWMSO 인사관리 상세 설명서

## 1. 메뉴 개요

`인사관리`는 현재 프로그램에서 가장 기능이 많은 메뉴입니다.
원본 기준으로 인사, 근태, 급여, 복지, 문서 기능이 한 화면 체계 안에 통합되어 있습니다.

## 2. 진입 방법

1. 좌측 사이드바에서 `인사관리`를 누릅니다.
2. 상단의 업무 공간을 먼저 선택합니다.
3. 그 다음 그룹별 탭을 선택합니다.
4. 일부 화면은 내부 보조 탭을 한 번 더 선택합니다.

## 3. 상위 구조

인사관리는 3개 업무 공간으로 구성됩니다.

1. `인력관리`
2. `근태 · 급여`
3. `복지 · 문서`

## 4. 업무 공간별 메뉴

### 4-1. 인력관리

1. `구성원현황`
2. `인사발령`
3. `포상/징계`
4. `교육`
5. `조직도편집기`
6. `스킬매트릭스`
7. `오프보딩`

### 4-2. 근태 · 급여

1. `근태`
2. `교대근무`
3. `근무표자동편성`
4. `연차/휴가`
5. `급여`
6. `간호근무표`
7. `공휴일달력`

### 4-3. 복지 · 문서

1. `건강검진`
2. `경조사`
3. `면허/자격증`
4. `의료기기점검`
5. `비품대여`
6. `사고보고서`
7. `계약`
8. `문서보관함`
9. `증명서`
10. `서류제출`
11. `캘린더`

## 5. 내부 보조 탭 구조

### 근태 화면 내부 탭

1. `근태 현황`
2. `연차소멸알림`
3. `지각조퇴분석`
4. `근무형태변경이력`
5. `조기퇴근감지`

### 급여 화면 내부 탭

1. `급여 메인`
2. `원천징수파일`
3. `4대보험 / EDI`

### 계약 화면 내부 탭

1. `계약 현황`
2. `계약서 자동생성`

### 캘린더 화면 구성

- `공유캘린더`
- `캘린더동기화`

## 6. 메뉴별 핵심 설명

### 구성원현황

- 직원 목록과 상세 인사 정보를 확인합니다.
- 인사관리의 기준 화면 역할을 합니다.

### 인사발령

- 인사 이동, 발령 관련 기록과 처리를 관리합니다.

### 포상/징계

- 포상 및 징계 이력을 기록하고 관리합니다.

### 교육

- 교육 계획, 이수, 대상자 관리에 사용합니다.

### 조직도편집기

- 조회용 조직도가 아니라 편집용 조직도 관리 기능입니다.

### 스킬매트릭스

- 직원 역량과 스킬 분포를 관리합니다.

### 오프보딩

- 퇴사 처리와 종료 체크리스트를 관리합니다.

### 근태

- 출퇴근, 월별 현황, 근태 분석을 관리합니다.

### 교대근무

- 교대 근무표와 스케줄을 관리합니다.

### 근무표자동편성

- 자동 편성 알고리즘 기반 근무표 기능입니다.

### 연차/휴가

- 휴가 신청, 승인 현황, 연차촉진 관련 업무와 연결됩니다.

### 급여

- 급여 처리 중심 화면이며 원천징수파일과 4대보험 EDI 기능이 내부 탭으로 연결됩니다.

### 간호근무표

- 간호 인력의 근무표를 관리합니다.

### 공휴일달력

- 휴일 기준 달력을 확인합니다.

### 건강검진, 경조사, 면허/자격증, 의료기기점검

- 복지, 법정관리, 점검성 업무를 처리합니다.

### 비품대여, 사고보고서

- 물품 대여 및 사고 보고 업무를 관리합니다.

### 계약, 문서보관함, 증명서, 서류제출, 캘린더

- 인사 문서 관리와 증빙 업무를 처리합니다.

## 7. 대표 사용 순서

### 직원 정보 확인

1. `인사관리`를 엽니다.
2. `인력관리` 업무 공간을 선택합니다.
3. `구성원현황`을 선택합니다.
4. 직원 목록에서 대상을 찾습니다.
5. 상세 정보를 확인합니다.

### 근태 분석 확인

1. `인사관리`를 엽니다.
2. `근태 · 급여` 업무 공간을 선택합니다.
3. `근태`를 누릅니다.
4. 내부 탭에서 `지각조퇴분석` 또는 `조기퇴근감지`를 선택합니다.
5. 기간과 대상자를 필터링해 확인합니다.

### 원천징수파일 확인

1. `인사관리`를 엽니다.
2. `근태 · 급여`에서 `급여`를 누릅니다.
3. 내부 탭에서 `원천징수파일`을 선택합니다.
4. 필요한 월 또는 기준을 선택합니다.
5. 파일 생성 또는 확인을 진행합니다.

### 계약서 자동생성

1. `인사관리`를 엽니다.
2. `복지 · 문서`에서 `계약`을 누릅니다.
3. 내부 탭에서 `계약서 자동생성`을 선택합니다.
4. 직원 또는 템플릿 기준을 선택합니다.
5. 생성 결과를 검토합니다.

## 8. 주의사항

- 인사관리는 권한별 노출 차이가 큽니다.
- 일부 메뉴는 화면상 보일 수 있어도 실사용 대상은 관리자나 인사 담당자입니다.
- `조직도편집기`는 조회용 조직도와 다르고, 수정 기능이 포함된 메뉴입니다.
