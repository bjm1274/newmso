# NEWMSO 사원용 기능 설명서

## 1. 문서 목적

이 문서는 일반 사원이 현재 원본 NEWMSO 프로그램에서 자주 사용하는 기능을 쉽게 이해할 수 있도록 정리한 안내서입니다.

## 2. 사원이 주로 사용하는 메뉴

| 메뉴 | 주 용도 |
| --- | --- |
| 내정보 | 출퇴근, 급여, 증명서, 서류 제출, 알림 확인 |
| 채팅 | 공지 확인, 1:1/그룹 대화 |
| 게시판 | 공지사항, 자유게시판, 경조사 소식 확인 |
| 전자결재 | 연차, 출결정정, 물품신청, 업무기안 |
| 추가기능 | 근무현황, 부서별 재고 등 보조 화면 |
| 조직도 | 필요 시 추가기능 또는 검색에서 진입 |

## 3. 내정보 사용법

현재 `내정보`에서는 아래 탭을 볼 수 있습니다.

| 탭 | 설명 |
| --- | --- |
| 프로필 | 내 기본 정보와 개인 대시보드 |
| 출퇴근 | 오늘 출퇴근 시간 확인 |
| 할 일 | 개인 메모형 할 일 관리 |
| 증명서 | 증명서 신청과 발급 이력 확인 |
| 급여 | 급여명세 확인 |
| 서류제출 | 회사 요청 서류 제출 |
| 알림 | 시스템 알림 확인 |

실무에서 가장 자주 쓰는 흐름은 아래와 같습니다.

### 출퇴근 확인

1. `내정보 > 출퇴근`으로 이동합니다.
2. 출근/퇴근 기록을 확인합니다.
3. 기록이 잘못되었으면 출결정정 요청을 진행합니다.

### 증명서 신청

1. `내정보 > 증명서`를 엽니다.
2. 필요한 증명서를 선택합니다.
3. 신청 후 처리 상태를 확인합니다.

### 서류 제출

1. `내정보 > 서류제출`로 이동합니다.
2. 요청된 서류를 업로드합니다.
3. 제출 후 관리자의 확인을 기다립니다.

## 4. 채팅 사용법

채팅은 사내 대화와 공지 확인에 사용합니다.

주요 기능

- 공지메시지 확인
- 1:1 채팅
- 그룹 채팅
- 이미지, 파일 전송
- 멘션
- 메시지 반응
- 북마크
- 스레드 답변

알아둘 점

- 공지메시지 방은 일반 직원이 자유롭게 쓰는 방이 아닐 수 있습니다.
- 현재 공지방 작성은 부서장 이상 직책 중심으로 제한됩니다.
- 공지방은 직원이 임의로 나갈 수 없도록 되어 있습니다.

## 5. 게시판 사용법

현재 게시판 종류는 아래와 같습니다.

| 게시판 | 설명 |
| --- | --- |
| 공지사항 | 회사 공식 공지 |
| 자유게시판 | 자유 글 공유 |
| 익명소리함 | 익명 의견 제출 |
| 경조사 | 경조사 소식 확인 |
| 수술일정 | 의료기관 일정 게시판 |
| MRI일정 | MRI 일정 게시판 |
| 직원제안함 | 제안 등록 |

일반 사원은 보통 공지사항, 자유게시판, 경조사 중심으로 사용하게 됩니다.

## 6. 전자결재 사용법

전자결재는 회사에 공식 요청을 올리는 메뉴입니다.

사원이 자주 쓰는 기능

| 기능 | 설명 |
| --- | --- |
| 기안함 | 내가 올린 문서 확인 |
| 작성하기 | 새 문서 작성 |
| 결재 캘린더 | 일정형 승인 확인 |

자주 사용하는 결재 종류

- 연차/휴가
- 출결정정
- 물품신청
- 수리요청서
- 업무기안
- 업무협조
- 양식신청

현재 화면상 `양식빌더`, `전자서명`, `직인관리` 같은 탭이 보일 수 있지만, 일반 직원은 보통 개인 결재 업무만 사용하면 됩니다.

### 연차 신청

1. `전자결재 > 작성하기`
2. `연차/휴가` 선택
3. 기간과 사유 입력
4. 결재선 확인 후 제출
5. 결과는 `기안함` 또는 `알림`에서 확인

### 출결 정정

1. `전자결재 > 작성하기`
2. `출결정정` 선택
3. 수정 사유 입력
4. 제출 후 승인 결과 확인

## 7. 추가기능 사용법

추가기능은 보조 화면 모음입니다.

현재 카드 구성

- 조직도
- 부서별 재고
- 근무현황
- 인계노트
- 퇴원심사
- 마감보고
- 직원평가

일반 직원은 일부 카드가 보이지 않을 수 있습니다.  
현재 제한 기능은 부서, 역할, 관리자 여부에 따라 숨겨질 수 있습니다.

추가로 아래 외부 링크가 함께 있습니다.

- `KM Park`
- `U+ 웹팩스`

## 8. 조직도 사용법

조직도는 현재 별도 메인 버튼보다 `추가기능 > 조직도` 또는 검색 결과 이동으로 들어가는 경우가 많습니다.

조직도에서 할 수 있는 일

- 사람 찾기
- 부서 찾기
- 회사별 조직 확인
- 팀별 보기

## 9. 자주 묻는 상황

### 메뉴가 안 보일 때

- 기능이 없는 것이 아니라 권한이 없을 수 있습니다.
- 부서장 또는 관리자에게 문의하면 됩니다.

### 급여가 안 보이거나 이상할 때

- `내정보 > 급여`를 먼저 확인합니다.
- 데이터 오류는 인사 또는 관리자에게 문의해야 합니다.

### 증명서나 서류 제출이 안 될 때

- 인사 담당자 또는 관리자에게 문의합니다.

### 결재가 안 올라갈 때

- 결재선, 입력값, 권한 문제일 수 있습니다.
- 관리자 또는 결재 담당자에게 문의합니다.

## 10. 문의 기준

| 상황 | 문의 대상 |
| --- | --- |
| 메뉴 미노출 | 부서장 또는 관리자 |
| 급여/인사 정보 오류 | 인사 담당자 |
| 증명서/서류 제출 문제 | 인사 담당자 |
| 결재 문제 | 관리자 |
| 로그인 문제 | 관리자 또는 마스터 운영자 |
