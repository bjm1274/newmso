## 2. 10년 데이터 보관 시 스토리지 비용 계산

### 2.1 월별 데이터 생성량 (50인 규모 병원)

```
근태 기록:        50명 × 20일 × 100byte = 100KB/월
전자결재 문서:    100건 × 50KB = 5MB/월
재고 로그:        500건 × 1KB = 500KB/월
메신저/채팅:      1,000건 × 5KB = 5MB/월
의료 기록:        1,000건 × 100KB = 100MB/월
검사 결과:        500건 × 50KB = 25MB/월
첨부 파일:        100개 × 500KB = 50MB/월 (1달 후 삭제)

월 총 데이터: 약 185MB
```

### 2.2 연간 및 10년 누적 용량

```
월 평균:          185MB
연간 누적:        2.22GB
10년 누적:        22.2GB

파일 삭제 정책 적용 (100MB 이상 1달 후 삭제):
- 첨부 파일 제외: 월 135MB
- 10년 누적: 16.2GB
```

## 3. Supabase 유료 플랜 비용 분석

### 3.1 스토리지 옵션

| 플랜 | 월 비용 | 포함 용량 | 초과 비용 | 10년 총 비용 |
|:---|:---:|:---:|:---:|:---:|
| **무료** | ₩0 | 1GB | 초과 불가 | ₩0 (1년만) |
| **Pro** | ₩25,000 | 8GB | GB당 ₩2,500 | ₩300,000/년 |
| **Business** | ₩100,000 | 100GB | GB당 ₩1,000 | ₩1,200,000/년 |

### 3.2 10년 데이터 보관 시나리오별 비용

**시나리오 1: 파일 1달 삭제 정책 (16.2GB)**
```
- Pro 플랜: 8GB 포함 + 8.2GB 초과
  월 비용: ₩25,000 + (8.2 × ₩2,500) = ₩45,500
  10년 비용: ₩45,500 × 120개월 = ₩5,460,000

- Business 플랜: 100GB 포함 (충분)
  월 비용: ₩100,000
  10년 비용: ₩100,000 × 120개월 = ₩12,000,000
```

**시나리오 2: 모든 파일 5년 보관 (약 11GB)**
```
- Pro 플랜: 8GB 포함 + 3GB 초과
  월 비용: ₩25,000 + (3 × ₩2,500) = ₩32,500
  5년 비용: ₩32,500 × 60개월 = ₩1,950,000
  10년 비용 (5년 후 안정화): ₩32,500 × 120개월 = ₩3,900,000
```

```

   - 5~10년 데이터

   - 비용: 월 ₩1,000 (매우 저렴)

총 월 비용: ₩31,000
10년 총 비용: ₩3,720,000
```
